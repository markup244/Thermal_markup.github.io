Viewing Material Properties

The properties of a selected material can be displayed in the View Existing Material Data dialog box.

Procedure

  1. Click the [...] button next to the dropdown list of materials to open the Material Library Selector dialog box.
  2. Select the material in either the Selected Items or Material Library panes, then click View to open the View Existing Material Data dialog box.
  3. After viewing the material properties, click OK to close the View Existing Material Data dialog box.
  4. Click OK to close the Material Library Selector dialog box.
