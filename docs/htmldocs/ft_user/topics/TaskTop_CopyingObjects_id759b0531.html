Copying Objects

You can make a single copy of one or more objects.

Procedure

To copy one or more objects to a different assembly or to a different location:

If you want to...

Do the following:

Copy to a different assembly in the data tree.

  1. Select the object(s) to be copied and press Ctrl+C.

  2. Select the new parent assembly and press Ctrl+V.

    The pasted object(s) are added to the bottom of the assembly in the same order they were prior to being copied.

Copy to a different location.

  1. Select the object(s) in the drawing board.

  2. Ctrl+drag the selected object(s) to the required location.
