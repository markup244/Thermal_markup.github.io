FloSCRIPT

Advanced users can run a FloSCRIPT file from MCAD Bridge.

XML script files are written for each session in the folder:

install_dir\flosuite_vversion\flotherm\WinXP\bin\LogFiles

Each file has a unique name, using the naming convention MCADLogFilenumber.xml.

Note:

Only the last five log files are retained when a new MCAD Bridge session is started. You can retain log files between sessions, however, you must prefix the filename so that it does not begin with the (case-insensitive) “MCADLogFile” string.

A FloSCRIPT file can be run from the GUI by choosing Macro > Play FloSCRIPT and then selecting the file, or from the command line, by specifying the file in a flomcad command.
