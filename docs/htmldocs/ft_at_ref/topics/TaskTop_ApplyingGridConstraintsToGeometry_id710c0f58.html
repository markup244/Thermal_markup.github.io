Applying Grid Constraints to Geometry

Grid constraints define the minimum grid requirement across selected geometry, that is, objects, assemblies, and cutouts on the solution domain.

Procedure

  1. Select the geometry in either the data tree or drawing board.
  2. Open the Attachments tab of the geometry property sheet. To apply grid constraints in different directions, expand the Grid Constraint, see Figure 1.
    Figure 1. Expanded Grid Constraints

  3. Make sure that the grid is visible in the drawing board view(s): key G.
  4. Either select an existing Grid Constraint attribute or create a new one.

    The Grid Constraint property sheet is displayed and the drawing board view is updated to show the effect of the grid constraint.

  5. If necessary, edit the Grid Constraint property sheet.

    As you make modifications to the property sheet you will be able to see the effect in the drawing board.
