Model Setup Tab - Gravity Section

To access: Click the Model Setup tab, then scroll down to the Gravity field.

Use this section of the Model Setup tab to activate or deactivate the effects of gravity. When activated, gravity can be simulated as a force acting on the fluid either parallel or angled to any of the coordinate directions.

Objects

Field

Description

Gravity

  • Off — Removes the effects of gravity from the solution.

  • Normal (default) — Activates gravity to align with one of the three coordinate directions selected from the popup.

  • Angled — Activates gravity in the direction set by vector resolutes.

Direction

(Normal) The choice is in the positive or negative X, Y, or Z directions.

Direction XN, YN, ZN

(Angled) Sets the resolute of the vector in the x-direction, y-direction, and z-direction respectively. The vector is normalized into a unit vector at the start of the next solve step.

For an example, see Angled Flow in the Simcenter Flotherm SmartParts Reference Guide.

Value

(Normal or Angled) The value of the gravity constant used for the program solution.

  • Automatic — The system default value of 9.81 ms-2.

  • User Specified — A user value.

Gravity (g)

(User Specified) Sets the Gravity value.
