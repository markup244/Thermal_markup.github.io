Saved Tables Property Sheet

To access: Expand the Saved Tables node of the Results tree and select a saved tables.

Use this property sheet to open, rename, or redefine a saved tables. A saved tables is a set of one or more results tables associated with one or more data tree objects.

Objects

Object

Description

Name

Editable name field identifying the saved tables.

Geometry

The name of the associated geometry.

Update

Redefines the saved tables to be those associated with the currently-selected geometry.
