Moving and Copying Library Items Between Library Folders

Library house-keeping tasks.

Procedure

You have a choice.

If you want to...

Do the following:

Move an item.

  1. Right-click the library item and select Cut.

  2. Right-click the destination folder and select Paste.

Alternatively, drag the item between library folders.

Copy an item.

  1. Right-click the library item and select Copy.

  2. Right-click the destination folder and select Paste.

Alternatively, hold down the Ctrl key and drag the item between library folders.
