Dynamic 2-Resistor Model

In this example, the Network Assembly includes three nodes: Junction, Case, and Board.

The external surface of the Case node is included as a collapsed cuboid on the top of the Case cuboid. The Board node is included as a collapsed cuboid on the bottom of the Junction node (see Figure 1).

Figure 1. 2-Resistor Model in the Data Tree

The drawing board views of the model are shown in Figure 2, where the Junction node is selected. In this model the junction and case nodes are set to have equal thickness. The cuboids representing the Junction and Case nodes are geometrically identical to those created by the Compact Component SmartPart, see Figure 1.
