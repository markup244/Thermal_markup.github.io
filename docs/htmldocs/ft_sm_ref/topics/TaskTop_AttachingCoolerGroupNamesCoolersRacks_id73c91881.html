Attaching Cooler Group Names to Coolers and Racks

When using Rack temperature to control Cooler airflow, at least one Cooler and one Rack must be assigned to the same Cooler Group.

Procedure

  1. Select the Cooler or Rack and open the Groups tab of the property sheet.
  2. Open the Cooler Group dropdown menu and either select an existing name or choose Create New to open the Create Group dialog box and enter a new name.
