Add a Solution Grid

During program solution, Simcenter Flotherm integrates the relevant differential conservation equations over each computational grid cell, assembling a set of algebraic equations which relate the value of a variable in a cell to its value in its nearest neighbor.

For example, temperature T shown in Figure 1 is related to temperatures T1, T2, T3, T4, T5, and T6 in the neighboring cells.
