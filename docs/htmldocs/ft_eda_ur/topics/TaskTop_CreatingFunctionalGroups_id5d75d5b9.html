Creating Functional Groups

You can create an empty functional group, or a functional group of existing components.

Procedure

  1. You have a choice:

    If you want to...

    Do the following:

    Create a new empty functional group ready to hold new components.

    1. Select the board to contain the functional group.

    2. Click the Create new Functional Group icon.

    The functional group is added to the data tree, ready to accept member components. An empty functional group does not appear in the 2D view because it is a reference origin to which components are added later.

    Create a new functional group from existing components.

    1. Select the components to be placed in the functional group

    2. Right-click over the components and choose Add to new Functional Group.

    The bounding box of a populated functional group is a black outline (see Figure 1) if the Display Functional Groups icon (Advanced Display Options toolbar) is active.
    Figure 1. Functional Group Bounding Box

  2. Move the location of the origin of a functional group to move all the components.
  3. Use the functional group property sheet to define the start X-Y location and the combined power output of the functional group.
  4. To swap a functional group with one held in the library, right-click the functional group in the data tree and choose Replace with library item, then use the Library Item Selector Dialog Box to find and load the required functional group.
