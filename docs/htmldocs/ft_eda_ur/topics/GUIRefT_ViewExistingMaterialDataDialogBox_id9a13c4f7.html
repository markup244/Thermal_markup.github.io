View Existing Material Data Dialog Box

To access: From the Material Library Selector dialog box, select a material type (either from the Selected items list or from the library), then click View.

Use this dialog box to view the properties of a selected material.

Objects

Read-only values for properties of the material.

Orthotropic materials have thermal conductivity defined along each axis: x, y and z.
