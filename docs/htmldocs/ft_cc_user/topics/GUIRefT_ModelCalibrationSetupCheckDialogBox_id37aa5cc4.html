Model Calibration Setup Check Dialog Box

To access: Calibration > Model Calibration then click Setup Check.

Use this dialog box to compare the parameter values of power, transient duration and initial temperature read from the T3Ster calibration file with those defined in the Simcenter Flotherm model, and make adjustments to the Simcenter Flotherm model.

Objects

Object

Description

Calibration Setup Check Table

Values of power, transient duration and initial temperature from the T3Ster calibration file and from the Simcenter Flotherm model. The third column indicates:

  • OK: the values do not need to be adjusted.

  • Warning: the model values may need to be adjusted to obtain good calibration results.

It is up to the user to decide whether to adjust the model values, small differences may be tolerable.

Note:

The transient duration of the model should be the same as, or longer than, the duration of the experiment (T3Ster).

Regenerate Time Patches

On by default. Time patches are regenerated based on a new transient duration.

Adjust

Click if you want the Simcenter Flotherm model values to match the T3Ster values.

Note:

Any existing scenarios are re-built and, therefore, solved results are invalidated.
