Solid Conductor Fluxes Results Table

To access: In Analyze mode, select one or more solid conductors (cuboids, prisms, tets, or inverted tets), or SmartParts built from primitives.

The table displays the thermal information for the selected solid conductors in the project. There is one row for each surface that has heat transfer (not insulated).

Note:

For interpretation of the results when cuboids are collapsed or overlap, see “Temperature and Heat Flow Values in Solid Conductor Fluxes Results Tables”.

Usage Notes

You can optionally display data for the constituent primitives of SmartParts, see “Expanding Results Tables to Show SmartPart Primitive Data”.

The information supplied does not cover thermal data for solids that connect directly to Compact Components. This information can be obtained from the Compact Component and Component Fluxes tables.
