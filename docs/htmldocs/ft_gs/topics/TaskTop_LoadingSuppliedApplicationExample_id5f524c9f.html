Loading a Supplied Application Example

Practical application examples are supplied with the installation, some of which include results.

Procedure

  1. Choose Project > New to open the New Project dialog, and then click the Application Examples tab, see Figure 1.
    Figure 1. Application Examples

    Examples that have included solved results have icons with plot color shading.

  2. Select an example and click OK.

    The project is opened.
