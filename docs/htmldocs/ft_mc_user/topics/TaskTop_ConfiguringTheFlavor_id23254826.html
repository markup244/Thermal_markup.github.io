Configuring the Flavor

Follow this procedure to add the required Simcenter Flotherm flavor data.

Procedure

  1. Copy the file <IGES install dir>/gif/USERFLAVOR.CON to the directory you start IDEAS from.
  2. Read the setup directions in this file and add the following flavor.
    FLOTHERM 
0 
1 
4 
7 1 2 9 9 9 9 9  
9 3 7 6 6 6 6 5 5 9 
3
    Note:

    If there were already extra flavors defined in the file, increment the first integer by one, otherwise add a 1 to the file before the FLOTHERM word, as the instructions in USERFLAVOR.CON describe.
