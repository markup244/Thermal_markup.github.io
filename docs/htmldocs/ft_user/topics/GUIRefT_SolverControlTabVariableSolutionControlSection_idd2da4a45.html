Solver Control Tab - Variable Solution Control Section

To access: Click the Solver Control tab, then scroll down to the Variable field.

Use this section of the Solver Control tab to control the solution of a field variable.

Description

The solution duration can be controlled by the false-time step, the termination residual, and the number of inner iterations.

Objects

Field

Description

Variable

Select the variable to which the following settings will apply. The variables available depend on:

  • The Type of Solution.

  • The Turbulence Model.

  • If the fluid density of any Fluids in the model is set to follow the Ideal Gas Law.

  • If Joule Heating is switched on.

See “The Solution Variables”.

Inner Iterations

Sets the maximum number of inner iterations on the linearized equation solver for the selected variable. See “Inner Iterations”.

False Time Step

(Not Pressure variable) Sets a transient-like parameter, that is used to damp or under-relax the solution of any variable except Pressure. Set to correspond to the timescale of the problem, for example, (typical length)/(typical velocity).

Type

  • Automatic — based on fans in forced flow cases and the EFCV in natural convection cases.

  • User Specified — enables you to override the automatic settings.

Damping Slider Bar

(Automatic) Use to change the automatic setting by either dividing or multiplying it by a factor of up to 20. Larger values give light damping, small values give heavy damping.

Value

(User Specified, read only if Automatic) User-specified value.

Termination Residuals

Sets the convergence tolerance, that is, the acceptable total error in solution for a variable.

Type

  • Automatic — calculated as:

    • For Pressure: 0.5% estimated typical mass flow.

    • For Velocities: 0.5% estimated typical momentum.

    • For Temperature: 0.5% of fixed heat source.

    • For Potential (Joule Heating on): 0.5% of the total current sources.

      Note:

      If the model contains only Fixed Value Potential electrical sources, then the estimated termination residual may be too conservative. You can change the default value and use monitor points to check convergence.

  • User Specified — enables you to override the automatic settings.

Factor Slider Bar

(Automatic) Use to change the automatic setting by either dividing it by up to a factor of 10, or multiplying it by a factor of up to 20.

Value

(User Specified, read only if Automatic) User-specified value.

Linear Relaxation

This is a legacy field. The field is only displayed if a pre-V10 project with a non-default setting for Linear Relaxation has been loaded.

Initial Variables

Sets initial values for individual field variables to be solved and stored. These initial values either help convergence in steady-state solutions or provide the starting conditions for transient calculations.

All Initial Values

Sets the method of initialization of all variables:

  • Automatic — generate values by program.

  • User Specified

  • Selected Solution Set — only active when Select has been used to load the results from another project. Sets the variable values to that of the loaded set.

When displaying Set Individually, at least one variable’s method of initialization differs from the others.

Initial Value

Sets the method of initialization of the currently-selected variable: Automatic, User Specified or Selected Solution Set.

Value

(User Specified) User-specified value.

Solution Set

(Selected Solution Set) The name of the loaded (selected) solution set.

Note:

Solve is denied if this field is empty when Select Solution Set is active, and Error message E/1027 is output during re-initialize. Select a different option in the Initial Value dropdown, or select a solution set. This applies to each variable.

Select button

(Selected Solution Set) Click Select to open the Solution Set Dialog Box to select the results data from another project as the initial values for the next solution run.
