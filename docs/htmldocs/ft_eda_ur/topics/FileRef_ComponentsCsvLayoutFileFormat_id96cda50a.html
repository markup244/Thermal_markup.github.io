Components CSV Layout File Format

A comma-separated text file generated by choosing File > Export > Export CSV.

Use this file to view or use component data in spreadsheet format, or to import component data to a model.

Format

A header row followed by a row for each component in the project.

Parameters

The row content is defined by the columns selected in the Select Column Types dialog box.
