Starting MCAD Bridge in Stand-Alone Mode

MCAD Bridge can be run as a stand-alone program on Microsoft® Windows systems.

Procedure

You have a choice:

If you want to...

Do the following:

Run MCAD Bridge from the Start menu.

Choose:

MentorMA > Simcenter Flotherm version MCAD Bridge

Run MCAD Bridge from any command window.

  1. Open a command window and change the directory to:

    install_dir\flosuite_vversion\flotherm\WinXP\bin

    where install_dir is the directory in which you have installed Simcenter Flotherm.

  2. Type the start-up string:

    flomcad.bat

    The flomcad command has optional parameters to start up with a loaded model.
