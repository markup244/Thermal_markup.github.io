Use With Care

By default, Simcenter Flotherm uses the wall functions to determine the heat transfer coefficient at the surface of an object as part of the standard solution. Surface exchange attributes overwrite this value by fixing the heat transfer coefficient to a surface or volume based method.

Using surface exchange attributes to specify heat transfer coefficients, and possibly reference temperatures also, is very advanced functionality which is not required in most electronics cooling simulations.

By specifying these parameters, instead of allowing the software to calculate them automatically, is dangerous as it pre-supposes a lot about the situation under study. In some cases this is justified but, unless the model very closely matches the set-up used to derive the specified values, it could be inaccurate.

Refer to Wall Functions in the Simcenter Flotherm Background Theory Reference Guide.
