Setting the Default Project

The default project is the project that opens when you start Simcenter Flotherm.

Procedure

  1. Press Ctrl+N to open the New Project dialog box.
  2. Select the template that you want to be the default project.
  3. Click Set Default.

    • If you only want to set the default but do not want to replace the currently loaded project, click Cancel.

    • To replace the currently loaded project with the default, click OK.

Results

The next time that you start up Simcenter Flotherm this template will be loaded by default.
