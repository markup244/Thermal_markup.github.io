PowerCoverageOption Method

Use to set the conductor coverage of power layers.

Prerequisites: None.

Usage

FlothermAddinControl.PowerCoverageOption(percent)

Description

Sets the percentage conductor coverage of power layers.

Only used when the assumed coverage option is active, see UseImageOption Method.

Arguments

  • percent

    A Real number between 0 and 100.
