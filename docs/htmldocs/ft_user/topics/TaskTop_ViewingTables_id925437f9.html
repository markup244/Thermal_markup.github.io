Viewing Tables

Tables are displayed in a pane within the Project Manager window, or in a dedicated window.

The number of significant figures of the values is defined in the Analyze Mode tab of the User Preferences dialog box.

Prerequisites

  • The Project Manager is in Analyze mode.

    By default, results tables are displayed in the Tables pane below the GDA.

Procedure

  1. To detach the Tables pane from the Project Manager, click on the header, drag the pane away from the Project Manager window, then release the mouse button.

    Tables are displayed in a separate window.

  2. To reattach a detached Tables window, drag the window to below the GDA until a blank pane appears, then release the mouse button.
  3. To re-open a Tables window or pane that has been closed, choose Window > Launch Tables.
