Exporting FLOPLC Files

You can export a *.floplc file containing the placement, component power dissipation and thermal data that can be read by supplied direct interface tools.

Restrictions and Limitations

Placement update into CR-5000 is not supported.

Procedure

Choose File > Export > Export FLOPLC and enter the filename and destination from the browser.
Note:

As Mentor Graphics Mechanical Analysis products identify placement files by the .floplc extension, when naming the file ensure you keep the default .floplc extension.

Results

The exported placement file can now be imported into Allegro or Board Station using the FloTHERM Interface > Import Placement Changes menu option in the interfaces supplied with EDA Bridge.
