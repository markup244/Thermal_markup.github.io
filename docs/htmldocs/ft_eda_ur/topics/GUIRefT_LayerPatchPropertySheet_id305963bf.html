Layer Patch Property Sheet

To access: Select a Layer Patch.

Use this property sheet to define the size, position, and percentage copper coverage of a layer patch.

Description

0.1 m

% Coverage

Data entry field referring to the area weighted averaged proportion of copper in that layer. The default % Coverage of 100% is suited for representing copper pads.

The % Coverage is represented in the display area by varying the patch color in graded steps from green (0%) to orange (100%).

100

Patch Material

Choice of Copper (Pure) or Copper (Aluminized).

Copper (Pure)
