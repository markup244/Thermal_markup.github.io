Batch Files

Batch files contain sequences of flotherm command lines.

File, flobatch.bat, is provided in folder install_dir\flosuite_vversion\flotherm\WinXP\bin.

This file is set up to accept the name of the project to be solved and, if so desired, the project to be used for the initialization, as command line input. The syntax is:

flobatch project_to_solve [project_to_initialize_from]

Save the batch file with the extension .bat into any directory.

Names are case-sensitive on UNIX systems.

For example (Windows), to solve two projects, My Project1 and My Project2, add the following lines:

call "C:\path\flotherm" -b \"My Project1\" 
call "C:\path\flotherm" -b \"My Project2\"

where path is:

install_dir\flosuite_vversion\flotherm\WinXP\bin
