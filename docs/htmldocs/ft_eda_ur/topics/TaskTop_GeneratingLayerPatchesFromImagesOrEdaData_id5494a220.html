Generating Layer Patches From Images or EDA Data

Monochrome images of PCBs, detailing copper and insulators in black and white respectively, can be used to quickly set-up representative layer patches. Using one of the supplied direct EDA interfaces will enable import of all board data, including image representation for each layer.

Procedure

  1. Right-click a Metallic layer in the tree and choose Process Layer from the popup menu.

    If File Name in the Layer property sheet has been set to the name of an image file, the Layer Trace Processing dialog box is opened, otherwise a file browser is opened to select the image before the dialog box is opened.

  2. Use the dialog box to adjust the representation.

    Image colors can be optionally inverted when the range is processed into layer patches.

  3. When finished, click Create Patches.
