Cost Function

A cost function is the value that can be minimized, maximized, or made a specified number by the sequential optimizer.

A cost function can use a single output variable, for example, solid cuboid temperature, or a combination of output variables.

The cost function is the weighted sum of output variable values:

Cost Function = W1R1 + W2R2 + ... + WnRn

Where Wn are the weighting factors, and Rn are the values of the output variables.

The optimum design is that with the optimum value of a set cost function. The optimum value of a cost function is the lowest or highest value of that function, depending on the way the function is expressed.

A cost target can also be defined, such that:

Cost Function = W × (R − Cost_Target)
