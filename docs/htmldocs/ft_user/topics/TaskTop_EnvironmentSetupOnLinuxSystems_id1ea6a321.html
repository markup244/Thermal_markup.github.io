Environment Setup on Linux Systems

Use the following procedure if you need to run Mentor Graphics tools which require the Simcenter Flotherm environment setup but Simcenter Flotherm has not been run.

Procedure

Run source setup in the following directory:

install_dir/flosuite_version/flotherm/LINUX/bin

source setup sets up the path, and environment variables required to run Simcenter Flotherm and its utilities, but does not run Simcenter Flotherm.
