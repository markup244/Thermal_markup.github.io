Grid Inflation

Use grid inflation to add extra grid cells to the sides of objects, thereby enabling closer monitoring near surfaces.

Default Inflation

If inflation is not specified to a side of a selected object, then, when you click the Inflate Grid icon or choose Grid > Inflate, a default inflation of 5% on the high and low coordinate side length, with a Minimum Number of 1 is applied by creating and attaching a grid constraint to the object. The name of the grid constraint is the object name appended with an underscore and letter, for example Cuboid:1_a.
