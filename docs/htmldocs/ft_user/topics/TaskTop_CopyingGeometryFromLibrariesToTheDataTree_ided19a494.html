Copying Geometry From Libraries to the Data Tree

Geometry objects (for example, fans) are provided in the installed libraries.

Restrictions and Limitations

  • When copying using the Transfer To Project menu option or by double-clicking:

    • If nothing in the data tree is selected, then the geometry will, by default, be copied to the root assembly.

    • If an invalid item is selected in the data tree, then nothing will be copied.

Procedure

You have a choice.

If you want to...

Do the following:

Copy the geometry by double-clicking (simplest).

  1. Select an assembly in the data tree into which you want the geometry to be copied.

  2. Double-click the geometry in the Library.

Copy the geometry by drag and drop.

  • Drag and drop the geometry from the Library to the data tree.

    The program prevents invalid dragging and dropping.

Copy the geometry using the context-sensitive menu.

  1. Select an assembly in the data tree into which you want the geometry to be copied.

  2. Right-click the geometry in the Library and select Transfer To Project
