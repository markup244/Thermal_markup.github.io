Copying Libraries

Library folders can be copied to create new libraries.

Prerequisites

  • The destination library folder must not be Read Only.

Procedure

  1. Right-click a library folder and select Copy.
  2. Right-click the destination library folder and select Paste.

    The copied folder is added to the Library tree.

    Tip

    The standard Ctrl+C, Ctrl+X and Ctrl+V shortcut keys can also be used to copy libraries.
