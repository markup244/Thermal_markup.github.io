Thermal Netlist/BCI-ROM Export Progress Dialog Box

To access: Displayed during the export of a thermal netlist or BCI-ROM data.

This dialog box shows solver progress data during an export, and closes when the export of data has completed.

Objects

Field

Description

Thermal Netlist/BCI-ROM Export

Read only. A percentage estimate of how far the export has progressed.

Start Time

Read only. The time the export was started.

Elapsed Time

Read only. The elapsed time during which the solver has been running.

Elapsed CPU

Read only. The recorded actual processing time of all utilized cores/CPUs
