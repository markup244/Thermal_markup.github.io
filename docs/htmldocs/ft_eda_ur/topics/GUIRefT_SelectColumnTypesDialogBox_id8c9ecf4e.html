Select Column Types Dialog Box

To access: File > Export > Export CSV

Use this dialog box to select which project information is to be included in an exported CSV file.

Material

As in the property sheets.

Usage Notes

This dialog box lists all the possible column types. Unchecked boxes exclude those columns from the CSV file.
