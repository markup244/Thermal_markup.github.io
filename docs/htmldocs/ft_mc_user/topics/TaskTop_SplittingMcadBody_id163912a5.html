Splitting an MCAD Body

Use this method to split an MCAD Body into two or more MCAD bodies.

Procedure

  1. Select the MCAD Body and click the Split Body icon or choose Tools > Split Body to open the Split Body dialog box.
  2. Use one of the following methods to mark the location(s) of split(s).

    If you want to...

    Do the following:

    Mark the location from a vertex.

    1. Click the Select Vertex button.

      The mouse pointer changes to a cross.

    2. Use the mouse to select the MCAD vertex.

    3. Select the Direction (plane) of the cut.

    Mark the location from a face.

    1. Click the Select Face button.

      The mouse pointer changes to a cross.

    2. Use the mouse to select the MCAD face.

    3. Select the Direction (plane) of the cut.

    Mark the location by specifying an orthogonal plane.

    1. Select the Direction (plane) of the cut.

    2. Either use the Position slider bar or enter an axial coordinate to specify the position.

      The slider bar limits correspond to the minimum and maximum coordinates of the bounding box of the selected body
  3. Click Slice to mark where the split is to occur.

    Figure 1 shows examples of slice marks (in yellow) that have been created from a vertex, from a face and by specifying a position.

    Figure 1. Examples of Slicing

  4. Click Split to split the MCAD body into smaller bodies.

Results

Separate MCAD bodies are created by splitting along slice lines. Figure 2 shows the examples of Figure 1 after splitting.
