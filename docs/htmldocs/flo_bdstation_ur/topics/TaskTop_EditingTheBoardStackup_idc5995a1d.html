Editing the Board Stackup

Editing of the board stackup to be used for Simcenter Flotherm PCB or EDA Bridge.

Prerequisites

  • If a board cross-section has been set up using Setup > Layer Materials, this information will be used to create the initial setup.

Procedure

  1. Open Board Station and load your design.
  2. Choose FloTHERM Interface > Board Stackup.
  3. Use the Board Stackup dialog box to set board layer characteristics.
