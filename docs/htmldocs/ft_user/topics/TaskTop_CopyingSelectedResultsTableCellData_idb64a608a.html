Copying Selected Results Table Cell Data

Copying table cell data to the buffer for pasting.

Procedure

  1. Select the cells to be copied.
  2. For steady-state analyses, press Ctrl+C , or right-click and choose Copy.
  3. For transient analyses, you have a choice:

    If you want to...

    Do the following:

    Copy the selected data for the currently-selected time step.

    Press Ctrl+C , or right-click and choose Copy.

    Copy the selected data for all saved time steps, or, in the case of Monitor Points, copy the selected data for all time steps.

    Choose Copy (with Transient Data).

Results

The data is now in the buffer and can be pasted into a spreadsheet or text document.

If saving data for multiple time steps, data for each time step is written on a separate row.
