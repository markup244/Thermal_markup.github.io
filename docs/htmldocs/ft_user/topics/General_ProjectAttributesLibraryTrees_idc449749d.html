Project Attributes/Library Trees

The Project Attributes/Library trees are on the right of the drawing board. The quickest way to display or hide these trees is to press F7. Each tree is displayed in a separate tab.

Attributes are used to define non-geometric properties which can be added (attached) to objects. The Project (Attributes) tree lists all attributes that are currently used on the project.

Refer to the Simcenter Flotherm Project Attributes Reference Guide.

The Library tree initially lists the start-up libraries supplied with the program, but these can be added to, see “Libraries”.
