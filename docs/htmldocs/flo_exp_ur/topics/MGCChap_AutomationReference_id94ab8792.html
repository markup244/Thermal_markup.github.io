Automation Reference

Xpedition PCB can be automated by scripts which use data objects, methods and so on in the data models.

The Simcenter Flotherm Xpedition Interface (provided by an add-in) also makes its data model available to users, and this is described in this chapter.

For information about Xpedition PCB add-ins, see Using Add-in Controls in the PCB Automation Reference manual supplied with Xpedition PCB.
