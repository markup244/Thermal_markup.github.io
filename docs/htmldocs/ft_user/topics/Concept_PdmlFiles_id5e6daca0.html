PDML Files

PDML (Physical Design Modeling Language) files are used to transfer information to and from the external file system for either the complete project or any selected assembly in the data tree.

By default, PDML files are binary, however, ASCII access can be made available with a special license key. Contact your local Simcenter Flotherm supplier for more information on the ASCII access license key if required.

Mentor Graphics uses PDML files to provide library objects for your modeling; some of these are available through starter libraries.

Other Mentor Graphics products, such as FloTHERM PACK and EDA Bridge, provide models for Simcenter Flotherm using PDML.
