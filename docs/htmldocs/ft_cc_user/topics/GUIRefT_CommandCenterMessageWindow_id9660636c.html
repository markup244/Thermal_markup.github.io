Command Center Message Window

To access: Window > Message Window

This window reports errors, warnings, and information generated by the Command Center.

Description

This window functions as the Project Manager Message Window but acts in the context of the Command Center. This window is totally independent of the Project Manager Message Window.

Objects

All objects and fields.
