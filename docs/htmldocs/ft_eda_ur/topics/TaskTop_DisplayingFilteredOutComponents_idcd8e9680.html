Displaying Filtered-Out Components

Filtered-out components can be made visible by changing the default method of coloring components.

Procedure

  1. Click the Color Components icon.
  2. Select Modeling Level.

Results

Filtered-out components are displayed in wireframe, for example, see Figure 1.
