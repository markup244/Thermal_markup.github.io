Examples of Open and Symmetry Boundaries

How symmetry can be used to simplify a model.

In Figure 1, (a) models an overall boundary open on three sides. As it is symmetrical, only half the model needs solving; this is achieved using a symmetry plane, (b).

Figure 1. Open and Symmetry Boundaries

Cutouts on the solution domain boundary can also be included in a symmetry model as shown in Figure 2.

Figure 2. Cutouts on the Solution Domain Boundary

Note:

The two outer surfaces of the cutout are irrelevant and do not need to be considered as Open or Symmetry.

The drawing board distinguishes between both types of boundary using color:

  • Dark Gray for Symmetry.

  • Yellow for Open.
