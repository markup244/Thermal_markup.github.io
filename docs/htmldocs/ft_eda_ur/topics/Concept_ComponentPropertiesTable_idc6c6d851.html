Component Properties Table

If components are present, a table of the component properties can be displayed (Component Table tab).
Figure 1. Component Properties Table

Click a header to sort the table by that value, click again to reverse the sort order.

You can edit any component in the model by using this table. There are two types of data entry cells:

  • Text entry boxes containing numerical values. Double-click the cell to enter edit mode.

  • Menu request boxes for modeling options. Left-click the table cell to open a dropdown menu of options.
