Topping Assemblies or Parts

When viewing long data trees, you can restrict (“top”) the tree view to show only an assembly or part. By Topping assemblies you can methodically step through all the assemblies in a particular path from the highest to the lowest level in the data tree. Topping is also reflected in the drawing board and so can be used to declutter views.

Procedure

  1. Select the object to be “topped”.
  2. Press F3 or choose View > Top.

Results

The selected object becomes the top item in the data tree as shown in Figure 1. All objects that are above the selected object are hidden from view.

Figure 1. Topping

  • To show the parent of the top item, press F5, or choose View > Up to Parent.

  • To display the whole tree, press F4, or choose View > Reset.
