Attaching an Attribute to an Object

Attributes must be attached to objects for them to take effect.

Attributes can be attached from the Project Attributes tree or directly from the Library tree.

Restrictions and Limitations

  • For a list of attributes and the objects to which they can be attached, see “Attribute Attachment to Modeling Objects”.

Procedure

Use one of the following methods:

If you want to...

Do the following:

Add one or more attributes directly to an object.

  1. Press F7 if the Project/Library pane is not open.

  2. Select one or more valid attributes in the Project Attributes tree or a single attribute in the Library tree.

    Multiple-select of library attributes is not allowed.

  3. Drag and drop the attribute(s) onto the object in the data tree.

Add a project attribute to an object via the object’s property sheet.

Note:

To add a library attribute to the project from the object property sheet, see “Adding an Attribute From a Library to the Project”.

  1. Open the Attachments tab of the object’s property sheet.

  2. Select from the available project attributes listed in the attachments dropdown list.
