Exporting a Legacy Results Table

Legacy results tables are not available as tables in the product window, but their data can be exported in CSV files.

Procedure

  1. Choose Analyze > Export Legacy Tables to open the Export Legacy Tables dialog box.
  2. Select the table(s) to be exported and click OK.

    The Select Export Folder dialog box is opened.

  3. Select a folder and click Select Folder.

Results

One or more CSV files are created in the selected folder.

The default CSV filenames are:

  • Geometry Model table - geometry_model.csv.

  • Grid tables:

    • baseGrid_axis-Grid_table.csv

    • localized object_axis-Grid_table.csv

      where:

      localized object is the name of an object that has had a localized grid applied.

      axis is X, Y, or Z.

    • (Transient solutions) time_grid_table.csv

  • Attributes table - object_attribute.csv.
