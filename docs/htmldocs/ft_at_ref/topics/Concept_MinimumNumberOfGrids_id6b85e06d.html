Minimum Number of Grids

Consider applying the grid constraints to a basic cuboid. By default, the program aligns the grid lines along the cuboid boundary.

Applying a minimum cell number of 5 ensures that, in areas of no interference from grids generated from other objects, four additional cells are added, as shown in Figure 1.

Figure 1. Minimum Number Constraint

Any grid interference is taken into account, and if sufficient grid already exists, then no action is taken.
