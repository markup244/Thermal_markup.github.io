Modeling Non-Uniform Dissipation Using Total Coverage Sources

Non-uniform power dissipation is modeled by defining the power output at selected rectangular locations across the face of the die.

Procedure

  1. In the Die Construction property sheet, select Total Coverage Sources Source Type.
  2. Click the Source Definition Click to Edit button.

    The Non-Uniform Total Coverage Source Definition dialog box is opened.

  3. Select the desired units for Power Unit.
  4. Depending on the how you want to enter data, choose one of the following methods.

    If you want to...

    Do the following:

    Enter data manually.

    1. Enter values for Number in Xo Direction and Number in Yo Direction.

      The number of table rows and columns changes.

    2. Enter power values in the table as required.

    Enter data by CSV file import.

    1. Click Import CSV File.

    2. Navigate to, and select, a valid CSV file and click Open.
  5. Click OK to save the data and close the Non-Uniform Total Coverage Source Definition dialog box.

Results

  • The Total Power value in the Construction tab of the Die property sheet is updated to show the total calculated power.

  • The drawing board shows the layout of the sources on the Die surface, for example, see Figure 1.
