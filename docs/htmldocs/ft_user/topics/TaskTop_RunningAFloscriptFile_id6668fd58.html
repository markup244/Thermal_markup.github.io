Running a FloSCRIPT File

FloSCRIPT files are XML script files that can run a sequence of operations.

Restrictions and Limitations

  • FloSCRIPT files are only applicable to one application: Project Manager, Command Center, MCAD Bridge, or EDA Bridge. If you try to run a file in a different application from which it was recorded, then an error message is generated.

Procedure

  1. Choose Macro > Play FloSCRIPT.

    The Play FloSCRIPT file browser opens. By default only *.xml files are listed.

  2. Navigate to the FloSCRIPT file and click Open.
    Note:

    FloSCRIPT files can also be run from the command line flotherm command.
