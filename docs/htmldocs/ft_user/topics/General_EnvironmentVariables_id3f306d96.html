Environment Variables

Simcenter Flotherm uses a number of environment variables to control the behavior of various aspects of the program.

CAUTION:

These variables are only recommended for the advanced user, and are not needed for the basic running of the software and may result in undocumented appearance, features, and behavior.

Defined indicates that any setting is valid; setting the value to TRUE in these cases will suffice.

Table 1. Environment Variables

Environment Variable Name

Value

Comment

ABUTTING_DISALLOWED

Defined

Fails the sanity check if objects or grid spaces are abutting.

CALIBRATION_EXPORT_DATA

Directory path

Defines the destination of exported Model Calibration transient data files. See Exporting Transient Data to T3Ster in the Simcenter Flotherm Command Center User Guide.

CCNUMBERTHREADS

Number

Sets the maximum number of threads (number of processors) for the parallel solver when running in batch mode or the Command Center.

COMPUTERNAME

None

Gets the computer name.

DEFAULT_ODB_PROPERTY

Enabled | property name

Reads component properties from ODB++ files when importing to EDA Bridge, and displays these in an expandable ODB Property field in the component property sheet.

If property name is specified then that name is the default that appears in the field.

FLO_CALIBRATION_LOAD_THREADS

Number

By default, half of the available hardware threads are used when T3Ster calibration data is loaded to the Command Center. You can set this environment variable to force the use of a specified number of threads.

FLOCENTRALDIR

Directory path

Specifies where the central library files are kept.

FLO_DOUBLEP

Defined

Sets the solver to use double precision instead of single precision.

Note:

If you only want to use the double-precision solver on a per project basis then set the Use Double Precision Solver option in the Solver Control tab, see “Double-Precision Solver”.

FLO_EMAIL

Email address

Requests solver status emails at end of a solve. used in conjunction with MAIL_HOST, which sets the server host.

FLO_FAN_NOISE_COEFF

Number

Overrides default coefficient (50) where Fan Noise (dB) = Nlog10 (derating factor).

FLO_FAN_POWER_EXP

Number

Specifies the power exponent in the calculation of derated fan power. Default value is 3. Derated Fan Power = Derating FactorN × Original Fan Power.

FLOFIXEDFLOWSOVERWRITE

Defined

Enforces hierarchical precedence when there are overlapping fixed flows, see “Context Rules for Overlapping Fixed Flows”.

FLO_HELP_PATH

File path and name

Defines the base directory location for the on-line help.

FLOHOME

Directory path

Specifies where the solution directory is kept (flouserdir) for FloVolunteer. Default defined in batch startup scripts.

FLO_IDF_CONFIG_FILE

File path and name

Defines the idf config file, for importing idf board data, complete with full path.

FLO_LANGUAGE

[jp | zh]

Alternative language displayed in the application windows:

  • jp − Japanese

  • zh − Simplified Chinese

FLOLOOK

[WINDOWS | WINDOWSXP | <ANYTHING ELSE>]

Specify look and feel of GUI. Default is WINDOWS on Windows, MOTIF on non-Windows (equivalent to <ANYTHING ELSE>).

FLOMCADKEEPLOGFILES

Defined

Instructs MCAD to keep any log files written when reading or writing CAD data files.

FLOMCADOPTIONDONOTUSETETS

Defined

Disable use of Tets during dissection

FLOMCAD_SET_COLOR_ON_TRANSFER

Defined

Specifies that the MCAD part color is transferred into the project

FLO_NUMBER_OF_THREADS

Number

Specifies the number of threads to run in the parallel solver.

FLO_OVERCOMPLEX_CELL_LIMIT

Number

Specifies a limit to the dissection of geometry by MCAD, thereby limiting computing resource requirements. Default 5000. If models are taking too long to dissect/decompose, reduce this number to obtain yield faster, but less accurate, results.

FLOPCM_MELT_BAND_MINIMUM

Number

When defining the Melt Temperature Band in Material attribute property sheets (Phase Change active), a minimum value of 0.1 degC is used regardless of what is entered in the property sheet. Use this environment variable to specify a different value (in units of degC).

FLO_SERVICE_PATH

Directory path

Override the location of required services.

FLO_SOLUTION_PRIORITY

[1 | 2 | 3 | 4]

Sets the runtime priority of the solver with the following specification: 1 - Idle/low, 2 - Normal, 3 - High, 4 - Realtime.

FLOPRODUCTCONFIG

File path and name

Specifies where the configuration file for the product is kept.

FLOPRODUCTDIR

Directory path

Specifies where the Mentor Graphics Mechanical Analysis executable files are kept.

FLOQUEUELOCATION

Directory path

Specifies the directory where the FloVolunteer queue is kept.

FLOQUEUENETWORKLAG

Number

Number of seconds to wait after saving a queue file over a network to make sure the file has been written and closed correctly.

FLOTINYMODEL

Defined

Enables sub-micron grid scales to be created.

Subsequent use of the double-precision solver is strongly recommended.

FLOUSERDIR

Directory path

Specifies the current solution data folder.

FLOUSERDIR_SAVE

Directory path

Specify the user directory for a Simcenter Flotherm Viewer Installation.

FLO_USE_V6_LINEAR_SOLVER

Defined

Force use of the v6 linear solver.

FLOVISHINTS

[QUICKDRAW | DISABLESTENCIL | DISABLESMOOTHSHADE | USE24BITRGB | USE12BITRGB | USE12BITMAP | USE8BITMAP | REPORTCONNECTION]

Various MCAD display options to do with render speed, shading, color depth, and the last option reports what video options are available.

FLOVOLUNTEERMAXJOBS

Number

Specifies maximum number of simultaneous jobs the Command Center should start.

Ignored when solving on Linux®1platforms, that is, no simultaneous processing of scenarios is possible.

FLOVOLUNTEERPARALLELTHREADS

Number

Species the maximum number of threads (= number of processors) for the parallel solver when running the FloVolunteer.

HOME

Directory path

Specifies users home directory.

HOMEDRIVE

Drive id

Specifies drive of user's home directory.

HOMEPATH

Directory path

Specifies path to user's home directory on drive HOMEDRIVE.

IDFSMALLEDGETOL

Floating point number

Tolerance size used when importing eda data. Any edge sizes below this tolerance are removed.

IV_SEPARATOR_MAX_CACHES

0

If you encounter memory problems when creating surface plots in Analyze mode, set the variable value to 0. This has the downside of generally slowing the plotting.

MAIL_HOST

Server name

Specifies mail server host for sending automatic email solve reports.

MAX_NUMBER_OF_OVERLAPS

Number. Default 2500.

The maximum number of overlapped localized grid spaces that will be resolved before a Warning message (W/1014) is output. This is a limit imposed to prevent excessive time being taken resolving overlapped localized grids.

MCADEXPORTVRML

Defined

Enables the Export VRML option in MCAD.

MGLS_LICENSE_FILE

File path and name

Specifies location of license file.

MENTORMA_PLUGINS

Directory path

Specifies the location of the file read/write plugins.

NUMBER_OF_PROCESSORS

Number

Specify the number of processors installed on the machine. Queried when running sequential jobs from Command Center.

OUTPUTEFGLOG

1

Activates an extra ASCII log file (viewf.log) of data from the EFG.

Warning: can create large files.

PATH

Directory path; Directory path;...

Path is retained and temporarily appended to when starting the software. Reset to retained value on exit.

SO_MATERIAL_DIR

Directory path

Specifies the directory containing information for visualization of materials.

SO_TEXTURE_DIR

Directory path

Specifies the directory containing information for visualization of textures.

TEMP

System defined

Temporary directory for all sorts of temporary operations.

TMP

System defined

Temporary directory for all sorts of temporary operations.

TMPDIR

System defined

Temporary directory for all sorts of temporary operations.

USERPROFILE

Directory path

Specifies the documents directory for the current Windows user.

XMLREADER_SCHEMA_FILE

Directory path

Specifies the directory containing the *.xsd files that define the valid syntax for XML files to be imported. This environment variable is set automatically when Simcenter Flotherm is started.
