Loading Data into a New Simcenter Flotherm PCB Project

Use this procedure when loading a layout for the first time.

Prerequisites

  • A Simcenter Flotherm PCB board data interface file, see “Generating a FLOEDA File”.

Procedure

You have a choice:

If you want to...

Do the following:

Load data from the command line.

  • Enter the following command:

 
runflopcb xyz.floeda

This invokes Simcenter Flotherm PCB with the board data interface loaded, creating a new project.

Load data from the GUI.

  1. Start Simcenter Flotherm PCB.

  2. Choose File > Import > Import FLOEDA and select the xyz.floeda file.
