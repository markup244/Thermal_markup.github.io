Recording a Sequence of Operations to a FloSCRIPT File

FloSCRIPT files are XML script files that can record a sequence of user operations.

Restrictions and Limitations

  • Restricted to the Project Manager, Command Center, MCAD Bridge, and EDA Bridge applications.

Procedure

  1. In the relevant application window, choose Macro > Record FloSCRIPT.

    The Record FloSCRIPT file browser opens. The default folder is

    install_dir\flosuite_version\flotherm\WinXP\bin

  2. Enter a filename for the FloSCRIPT file. Optionally, navigate to another folder. Click Save.
  3. Start performing the operations you want to record.

    You can pause (Macro > Pause FloSCRIPT) and resume (Macro > Resume FloSCRIPT) recording at any time.

  4. When finished, choose Macro > Stop FloSCRIPT.
