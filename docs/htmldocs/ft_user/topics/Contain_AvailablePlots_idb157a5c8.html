Available Plots

The profile plots that are generated depend on the solution type: steady-state or transient. A monitor point profile plot is generated if there is one or more monitor points. You can create a (distance) plot of results between two points in the solution domain.
  • Residuals v Iteration Plot (Steady State)
  • Residuals v Iteration Plot (Transient)
  • Residual Iterations v Time Plot (Transient)
  • Residuals v Time Plot (Transient)
  • Monitor Points v Iteration Plot (Steady State)
  • Monitor Points v Time Plot (Transient)
  • Results v Distance Plot
