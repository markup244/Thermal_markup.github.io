Importing T3Ster Files

Files generated by Mentor Graphics T3Ster-Master software can be imported into assemblies.

Procedure

  1. In the data tree, right-click the assembly which you want to hold the imported geometry and choose Import Assembly > T3Ster. A file browser is opened with a *.xCTM filter.

    A *.xCTM file is a file representing a compact thermal resistor/capacitor model of an IC package, generated by Mentor Graphics T3Ster-Master software.

  2. Select the file and click Open.

    A Network Assembly object is added to the assembly.
