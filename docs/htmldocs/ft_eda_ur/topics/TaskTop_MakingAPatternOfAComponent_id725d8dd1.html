Making a Pattern of a Component

Cloning components across the board in two dimensions.

Procedure

  1. Select the object(s) to be patterned.
  2. Choose Edit > Pattern to open the Pattern Selected Items dialog box.
  3. Define the number of copies and their separation in one or both directions.
