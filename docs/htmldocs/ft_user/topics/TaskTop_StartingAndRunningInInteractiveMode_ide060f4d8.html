Starting and Running in Interactive Mode

Running interactively enables setup, flow calculation, and results analysis to be completed in the same program session.

Restrictions and Limitations

  • Simcenter Flotherm can only run in interactive (GUI) mode on Windows platforms.

Procedure

  1. To start the GUI from the task bar, choose Start > All apps > MentorMA > Simcenter Flotherm version > Simcenter Flotherm version.
  2. (Optional) Create a shortcut to the Simcenter Flotherm batch file and use that:

    install_dir\flosuite_vversion\flotherm\WinXP\bin\flotherm.bat

    whereinstall_dir is the directory in which you have installed Simcenter Flotherm.
