Uniting, Intersecting, and Subtracting Multiple Bodies

Options are available for combining multiple objects.

Procedure

  1. To unite all selected intersecting bodies into one body, select the bodies then choose Tools > Unite Bodies. Figure 1 shows a typical result.
    Figure 1. Three Bodies United

  2. To deletes all but the intersecting regions of a number of bodies, select the bodies then choose Tools > Intersect Bodies. Figure 2 shows a typical result.
    Figure 2. Two Bodies Intersected

  3. To subtracts all selected intersecting bodies from the first selected body, select the bodies then choose Tools > Subtract Bodies. Figure 3 shows a result where three bodies have been replaced by a single body. The middle body was the first to be selected, subsequent selected bodies have been subtracted from that body.
    Figure 3. Subtracted Bodies
