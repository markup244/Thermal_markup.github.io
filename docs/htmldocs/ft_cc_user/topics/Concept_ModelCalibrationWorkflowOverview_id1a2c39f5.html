Model Calibration Workflow Overview

The Command Center can import data from T3Ster Master v2.4 and higher and optimize a Simcenter Flotherm model to produce a simulated structure function that closely matches a T3Ster structure function.

For a detailed description of the procedure within Simcenter Flotherm, see “Calibrating a Model Using Data From T3Ster”:

  1. The thermal response of the device is measured using T3Ster.

    T3Ster uses an electrical test method, defined by JESD 51-1, to measure the junction temperature.

  2. A “Best-Guess” model is created in Simcenter Flotherm and the Base Project solved for a transient analysis.

  3. A T3Ster calibration file is imported and is compared with the Simcenter Flotherm model.

  4. Command Center input variables (typically dimensions and materials) are selected and Design of Experiments scenarios created.

  5. The scenarios are solved and the model settings optimized to match the T3Ster-measured structure functions.

When successful, the result is a validated 3D detailed device model, suitable for use in any transient or steady state analysis.
