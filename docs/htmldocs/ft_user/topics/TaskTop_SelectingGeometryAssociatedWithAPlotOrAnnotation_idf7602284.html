Selecting Geometry Associated With a Plot or Annotation

Plots and annotations created when geometry is selected are associated with the selected geometry. Subsequently the geometry may become deselected. This task describes how to reselect the geometry.

Restrictions and Limitations

  • Applies only to plots and annotations that are associated with geometry.

Procedure

  1. Right-click a plot or annotation node in the Results Tree.
  2. Choose Select Associated Geometry.

Results

The associated geometry is selected in the data tree and in the GDA.
