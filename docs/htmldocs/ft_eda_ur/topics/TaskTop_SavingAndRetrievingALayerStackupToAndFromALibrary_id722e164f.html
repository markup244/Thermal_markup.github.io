Saving and Retrieving a Layer Stackup to and From a Library

The layer definition for a board can be saved to, and retrieved from, a library.

Procedure

  1. Right-click the layer stackup node in the data tree.
  2. Choose Save To Library.

    The Library Selector dialog box is opened.

  3. Select a library and click OK.

    Provided the library has write-access, the layer stackup is written to the library.

  4. To retrieve the layer stackup, select an existing layer stackup node in the data tree, right-click and choose Replace with library item.

    The Library Selector dialog box is opened.

  5. Open the relevant library, select the layer stackup and click OK.

    The existing layer stackup will be over-written by the library layer stackup definition.
