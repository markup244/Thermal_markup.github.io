Selecting Table Cell Data

For operations on the table data, such as data export and row hiding, selection of rows, columns, or cells is required.

Procedure

To select a cell, click in the cell.

Other selection options are described below:

If you want to...

Do the following:

Select a row.

Click a row header cell.

Select a column.

Click a column header.

Select the whole table.

Click the top-left corner of the table.
Note:

You can multi-select using Ctrl+click and Shift+click.
