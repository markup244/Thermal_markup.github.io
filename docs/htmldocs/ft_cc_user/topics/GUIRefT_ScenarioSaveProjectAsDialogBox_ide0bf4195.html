Scenario Save Project As Dialog Box

To access: Select a scenario, then choose Scenario > Save As

Use this dialog box to save the selected scenario project as a new Simcenter Flotherm project.

Objects

Field

Description

Project Name

The name of the project that will be created from the currently-selected scenario.

The default name assigned is:

base project name_scenario name

but this can be edited by over-typing.
