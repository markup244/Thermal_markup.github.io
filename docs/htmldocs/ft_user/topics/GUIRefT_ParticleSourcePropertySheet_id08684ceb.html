Particle Source Property Sheet

To access: Expand the Particles node of the Results tree and select a particle source.

Use this property sheet to define particle sources, particles, and particle streamlines. A particle source is a point from where particles originate. A particle streamline is a representation of the path that particles take through the domain.

Description

The property sheet has two tabs.

Objects

Definition Tab

Defines the particle sources and streamlines, see “Particle Source Property Sheet - Definition Tab”.

Appearance Tab

Defines the appearance of particles, see “Particle Source Property Sheet - Appearance Tab”.
