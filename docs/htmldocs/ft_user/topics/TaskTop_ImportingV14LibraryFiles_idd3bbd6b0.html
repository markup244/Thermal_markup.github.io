Importing V1.4 Library Files

Simcenter Flotherm version 1.4 library files can be imported into assemblies.

Procedure

  1. In the data tree, right-click the assembly to hold the imported geometry and choose Import Assembly > V1.4 File.
  2. In the file browser, select the required Simcenter Flotherm V1.4 *.lib or *.LIB file (the default file filter is for *.??b files) and click Open.

Results

The contents of an imported .lib file are added to the selected assembly. The imported items are named as the library file without the .lib extension and appended with a sequence number.

For further information on file conversion details, contact Mentor Graphics customer support.
