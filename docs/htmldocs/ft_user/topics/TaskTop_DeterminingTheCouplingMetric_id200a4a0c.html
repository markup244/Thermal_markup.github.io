Determining the Coupling Metric

The coupling metric, or the measure of the amount a zoom-in region changes the results, can be obtained using the project notes.

Prerequisites

  • A zoom-in project has been created, see “Creating a Zoom-In Project”.

Procedure

  1. Choose Project > Load to open the Load Project dialog box, select the project.
  2. Click Notes to view solution data saved from the original project.

    An example is shown in Figure 1.

  3. Solve the zoom-in project and compare the results with the data written to the project notes.

    If no changes are made to the zoom-in model, the data should be identical.

Results

The net mass and heat flows of each face of the zoom-in region can be obtained by summing the columns of the cutouts result table and comparing the values to their equivalent within the project notes.
