Exporting STEP Files From Pro/ENGINEER

Use the following procedure to export STEP files from Pro/ENGINEER for import into MCAD Bridge.

Procedure

  1. Choose File > Export > STEP to display the Export STEP dialog box (Figure 1).
    Figure 1. Export STEP Dialog Box

  2. Accept the default filename
  3. Ensure that the Export STEP dialog box has Solids selected and that Shells is not selected (it is by default).
