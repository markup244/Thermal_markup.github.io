Importing Network Assemblies From JEDEC Part Model XML Files

Network Assembly SmartParts can be created by importing JEDEC xml files.

Prerequisites

  • An xml file that conforms to the ThermalSection of the JEDEC XML Schema, as described in “JEDEC Part Model XML Files (ThermalSection)”.

Procedure

  1. In the data tree, right-click the assembly which you want to hold the imported geometry and choose Import Assembly > JEDEC Part Model Thermal.

    A file browser is opened with a *.xml filter.

  2. Select the file and click Open.

Results

The ThermalSection lines of the xml file are used by Simcenter Flotherm to create Network Assemblies, Network Nodes and Network Cuboids. All other lines of the file are ignored.

If there is no ThermalSection in the imported file, then a Warning message is output and nothing is created.
