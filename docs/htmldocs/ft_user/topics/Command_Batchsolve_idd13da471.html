batchSolve

Runs on Linux platforms.

Solves a Simcenter Flotherm project, described by a FloXML or PDML file, in a batch run.

Usage

batchSolve file [options]

Arguments

file

Name of a Project FloXML or PDML file to be solved.

[options]

The same options as for the flotherm command, except for the following:

  • -b

    This is not required because batchSolve always operates as a batch command.

  • -c

    Ensure that the base case is fully solved before using this option to solve all the unsolved Command Center scenarios for the project.

    The FLOVOLUNTEERMAXJOBS environmental variable is ignored; only one scenario is solved at a time.

  • -env

    This is a Windows-only option. To set up the path and environment variables required to run Simcenter Flotherm and its utilities, run source setup, as described under “Environment Setup on Linux Systems”.

Description

A typical installation puts batchSolve in directory:

/opt/MentorMA/flosuite_vversion/flotherm/LINUX/bin
