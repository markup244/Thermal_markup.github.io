Temperature Dependent Conductivity of Fluids

Temperature-dependency of conductivity is rarely required for dilute gases such as air.

CAUTION:

You should not set Temperature Dependent values unless advised to do so by Mentor Graphics.

However, some liquids have a more critical dependence between temperature and conductivity. For example, water has the variation shown in Figure 1 for the temperature range 270-to-370K.

Figure 1. Temperature Dependent Conductivity

To apply this variation with temperature, set the Fluid attribute property sheet values to:

  • Conductivity Type = Temperature Dependent

  • Conductivity = 0.58 W/(mK)

  • Coefficient = 0.0009 W/(m K^2)

  • T ref = 273 K
