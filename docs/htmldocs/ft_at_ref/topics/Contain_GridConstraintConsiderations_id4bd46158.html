Grid Constraint Considerations

Grid constraints allow you to attach minimum grid requirements to an item of geometry to ensure adequate grid coverage wherever it is located in the calculation domain.

If a grid constraint is applied that requires more coverage than the current grid provides, then the program adds more grid cells. However, if the grid constraint specifies less coverage than actually exists, then no action is taken. The grid constraint does not remove grid, it only ensures adequate coverage.

Note:

Grid constraints always try and save a uniform grid over the object to which they are applied, taking into account any existing grid.
