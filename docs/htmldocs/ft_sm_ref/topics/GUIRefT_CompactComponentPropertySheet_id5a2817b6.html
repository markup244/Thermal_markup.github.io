Compact Component Property Sheet

To access: Select a Compact Component SmartPart in the model.

Use this property sheet to define the properties of the Compact Component (compact resistor package).

Description

The Balls/Pins tab is not available if Peripheral package type has been selected in the Model tab.

The Leads tab is not available if Area Array package type has been selected in the Model tab.
