Export Legacy Tables Dialog Box

To access: Analyze > Export Legacy Tables

Use this dialog box to select which legacy results tables to export as CSV files.

Objects

Object

Description

Geometry Model

Creates a file named geometry_model.csv. See “Geometry Model Results Table CSV File”.

Grid

Creates base grid files named baseGrid_axis-Grid_table.csv. If there are objects with localized grid, also creates files named localized object_axis-Grid_table.csv. See “Base Grid and Localized Grid Results Tables CSV Files”.

For transient solutions, creates a file named time_grid_table.csv. See “Transient Grid Results Table CSV File”.

Attributes

Creates a file named object_attribute.csv. See “Object/Attributes Results Table CSV File”.
