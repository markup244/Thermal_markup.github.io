Inserting After a Scenario Project in the Scenario Table

You can insert a new scenario project after the currently selected project which is a copy of the selected project.

Procedure

  1. Open the Scenario Table and right-click in the scenario title cell.
  2. Choose Insert After.

Results

A new scenario with the same values as the selected scenario is inserted after the selected scenario. If subsequent scenarios have default names, for example “Scenario n” then these are renumbered.
