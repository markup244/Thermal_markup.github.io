Maintaining Grid Constraints Between Items

Use this procedure when you want to maintain a number of grid cells between a pair of items.

Procedure

  1. Add a block to the assembly containing the boards and apply a grid constraint to this block.
  2. Deactivate the block so that only the grid associated with it remains.

Results

When the board assembly is moved, then both of the boards and the deactivated block move, maintaining the grid, see Figure 1.
