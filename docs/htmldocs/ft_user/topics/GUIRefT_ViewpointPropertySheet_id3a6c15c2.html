Viewpoint Property Sheet

To access: Expand the Viewpoint node of the Results tree and select a viewpoint.

Use this property sheet to view the position and orientation of a viewpoint, and to jump to a viewpoint.

Objects

Object

Description

Name

The name of the viewpoint. Default names are Viewpoint n.

Camera Type

Sets the type of view:

  • Perspective − adjusts for the effects of perspective

  • Orthographic − no rules of perspective are applied

Height

The height of the camera view.

Position X/Y/Z

The camera location.

Angle of Axis X/Y/Z

The camera lens direction.

Angle of Orientation

The rotational angle of the camera about the axis of the lens. A value of 180 deg turns the camera upside down. A value of 360 deg is equivalent to a value of 0 deg.

Jump to Viewpoint button.

Display the model from the viewpoint. Alternatively, you can double-click the viewpoint in the Results tree.
