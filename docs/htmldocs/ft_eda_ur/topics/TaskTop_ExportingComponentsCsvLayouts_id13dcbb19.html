Exporting Components CSV Layouts

You can export board design information in comma separated variable files.

Restrictions and Limitations

  • Material names will be exported automatically into Components CSV files, where applicable.

  • When exporting a 2R component, if the simple variation of the component has non-default material referenced, this will not be preserved.

Procedure

  1. Choose File > Export > Export CSV to call the Select Column Types Dialog Box and choose the information to be stored.
  2. Use the file browser to choose the filename and destination.
