Bn and Sc

Normalized parameters that give an indication of thermal bottlenecks and paths.

Bn and Sn are not normally stored, but can be if the Bn and Sc check box in Stored Variables section of the Model Setup tab is checked. This check box only becomes active when Heat Fluxes and temperature gradient (GradT) values are stored.

Bn

is a normalized parameter that indicates thermal bottlenecks

Sc

is a normalized parameter that indicates possible path of heat flow

These values are available in Analyze mode when stored.
