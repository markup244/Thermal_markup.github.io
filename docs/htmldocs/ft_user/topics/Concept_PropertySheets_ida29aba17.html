Property Sheets

Property sheets are the standard user interface for specifying values and parameters.

A property sheet is opened when you select an object in the data tree or a project attribute. Most of the object property sheets have more than one tab.

Name Fields

Up to 32 characters may be entered in Name fields.

New geometry objects are supplied with a default name corresponding to the object type and if objects of the type already exist, ‘:n’ is appended to the name, where n is a sequence number, for example, Cuboid:1.

The name of an object can also be changed by double-clicking the object in the data tree, which puts the name into edit mode.

Units Dropdown List

The Units dropdown list enables you to override the default settings for the current property sheet.

Figure 2. Dimension Units Dropdown List

The values do not change, they are displayed in different units. For example, 0.1 m is displayed as 10  cm if the units are changed from meters to centimeters.
