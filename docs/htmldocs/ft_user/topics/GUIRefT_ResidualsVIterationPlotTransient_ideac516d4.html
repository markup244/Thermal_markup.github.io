Residuals v Iteration Plot (Transient)

To access: Profiles > Create Plot > Residuals v Iteration. A default plot is created during a transient solution.

Use this plot to view the convergence during a particular time step of a transient solution.

Description

Iterations for a selected time step are displayed.

Selection of the time step is from the Settings tab of the plot property sheet.

Figure 1. Docked Transient Solution Plot of Residuals v Iteration for a Time Step

Objects

Context-Sensitive Menu

Choose Edit Settings to open the Settings tab of the plot property sheet.
