Configuring Processor Resources for Solving Scenarios on a Local Machine

You can use solve scenarios in parallel and use multiple processors to solve each scenario.

Procedure

  1. Choose Edit > User Preferences to open the Command Center User Preferences dialog box.

    The Solver Configuration tab is open by default and the selected Run Solvers On default is Local Machine.

  2. Enter the number of scenarios that you want to be solved in parallel in the Concurrent Solvers field.
  3. Enter the number of processors to use to solve each scenario in the Number Of Processors To Use field.
    Note:

    Your local machine should have at least (Concurrent Solvers * Number Of Processors To Use) processors to avoid overloading.

  4. Click OK to save the settings and close the dialog box.

    The configuration will be used the next time you solve.
