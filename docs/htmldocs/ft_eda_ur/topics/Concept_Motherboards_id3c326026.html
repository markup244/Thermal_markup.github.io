MotherBoards

All EDA Bridge projects have a MotherBoard, which is the root object onto which you add objects.

By default, when a new project is opened, a MotherBoard with a group of four layers is created.

The following geometry can be added to a MotherBoard: Layers, DaughterBoards, Rectangular Components, Functional Groups, Rectangular Cutouts, Cans and Placement Keepout Regions.
