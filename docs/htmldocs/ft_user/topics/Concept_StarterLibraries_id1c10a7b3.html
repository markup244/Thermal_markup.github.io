Starter Libraries

The libraries are initially stocked with standard entries to get you started, but you can build the libraries to suit your own requirements.

Organizations can also set up their own standard entries to be shared among their engineering teams.

Table 1. Starter Libraries

Library

Contents

ATX Library

Different styles of chassis and equipment suitable for ATX_Part, SFF_Part_Library and microATX_Part_Library based form factors.

Blowers

Libraries of blowers by the following suppliers: EBM, Sanyo Denki and YS Tech USA.

Component Starter Pack

Different types of 2 Resistor Compact Components.

Data Center

APC, Generic, and Gilberts data center components.

  • For a description of the contents of the Generic library, see “Generic Data Center Library”.

  • For a description of the Gilberts GF Series Floor Grille, see “Gilberts Data Center Library”.

Fans

Libraries of axial fans by the following suppliers: Comair Rotron, Delta, ETRI, Micronel, NMB, Papst, and Sanyo Denki.

Filters

Libraries of resistance attributes representing the flow resistance of UAF foam filters (Planar), and libraries of assemblies containing resistances representing UAF foam and polyester media filters.

Simcenter_Flotherm_Libraries

Libraries for each attribute type, containing commonly used attributes pertaining to electronics.

Heat_Sinks

Subdirectories for models of Alpha Novatech, HS Marston, and IERC heat sinks.

The IERC ***README-NOTE assembly is an assembly that only contains a note about the IERC heat sinks. To read the note, copy the assembly into your project, then open the Notes tab of the property sheet.

Interface_Materials

Libraries of surface finishes representing Bergquist, Chomerics, Dow Corning, and Thermagon interface materials. The libraries are further subdivided into interface types.

Materials

Links providing quick access to the Materials library files listed under Simcenter Flotherm_Libraries.

Thermo Electric Coolers

Models of Marlow and Mercor TECs.

Tutorial Library

Contains an assembly, cbga1, as an example of a detailed component.
