Exporting Projects in SAT or IGS Files

Projects can be can be exported to external disk files.

Procedure

  1. You have a choice:

    If you want to...

    Do the following:

    Export an assembly as an ACIS *.sat file

    • Right-click the project name at the top of the Project Manager data tree and choose Export Project > SAT.

    Save an MCAD assembly as a *.igs format file

    • Right-click the project name at the top of the Project Manager data tree and choose Export Project > IGES.
  2. Navigate to the destination folder and click Save.

Results

  • Exported *.sat files do not include the assembly hierarchy, object names, or units.

  • Exported *.igs files do not include the assembly hierarchy, object, or names.
