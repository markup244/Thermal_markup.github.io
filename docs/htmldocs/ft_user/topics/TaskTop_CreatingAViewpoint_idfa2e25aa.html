Creating a Viewpoint

Save the current view of the GUI for later recall.

Prerequisites

  • The Project Manager is in Analyze mode.

Procedure

  1. Change the view of the GUI to one that you want to save.
  2. Click the Save Current Viewpoint icon or right-click the Viewpoint node of the Results tree and choose Save Current Viewpoint.

Results

The first viewpoint created is named Viewpoint. Subsequent viewpoints are named Viewpoint n.
