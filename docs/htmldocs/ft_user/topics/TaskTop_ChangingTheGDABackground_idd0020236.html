Changing the GDA Background

The GDA can have a uniform or a top-to-bottom gradient background.

Procedure

  1. Choose Edit > User Preferences to open the User Preferences dialog box.
  2. Open the Drawing Board tab.
  3. You have a choice:

    If you want to...

    Do the following:

    Have a uniform colored background.

    1. Leave the Gradient Background check box unchecked.

    2. Click the Background Color palette button to open the Modify Color dialog box.

    3. Use the Modify Color dialog box to pick an existing basic or custom color, or to define a new color

    4. Click OK to close the Modify Color dialog box.

      The color of the Background Color palette button changes to the newly selected color.

    Have a gradient background.

    1. Check the Gradient Background check box.

    2. Click the Top Background Color palette button to open the Modify Color dialog box.

    3. Use the Modify Color dialog box to pick an existing basic or custom color, or to define a new color

    4. Click OK to close the Modify Color dialog box.

      The color of the Top Background Color palette button changes to the newly selected color.

    5. Repeat Steps 2 to 4 to change the Bottom Background Color palette button.
  4. Click OK to save your changes and close the User preferences dialog box.
