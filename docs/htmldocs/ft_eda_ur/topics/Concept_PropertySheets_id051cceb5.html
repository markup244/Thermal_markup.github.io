Property Sheets

When you select an object when the tree view is shown, then the property sheet for the object is displayed beneath the data tree.

When two or more objects of the same type are selected, data can be entered once and applied to all the selected items. Settings that are different are left blank and have to be changed individually.

Figure 1. Property Sheet Setting for Multiple Objects

There are two types of data entry fields in the property sheets:

  • Text entry boxes for numerical values

  • Menu request boxes for modeling options

In both cases, left-click to activate the entry box.

  • For text boxes, the text insertion mark appears ready to change the name or value.

  • For menu request boxes, a choice of modeling options is displayed, see Figure 2.

    Figure 2. Property Sheet Menu Request Box

Construction notes can be added for all objects except layer patches and cutouts. Clicking the [...] button (see Figure 3) opens the Edit Text Dialog Box for adding notes.
