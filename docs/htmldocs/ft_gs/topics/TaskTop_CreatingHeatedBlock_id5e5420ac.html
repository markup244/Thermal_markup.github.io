Creating a Heated Block

A cuboid is added with an attached Material attribute to represent a heated block.

Procedure

  1. Select Root Assembly and add another cuboid.
  2. Make the following changes to the property sheet of the new cuboid:

    • Rename the object to Heated Block

    • Set Position to (mm) X = 35, Y = 120, Z = 140

    • Set Size to (mm) X = 5, Y = 40, Z = 40

  3. Expand the Attributes library down to Materials > Ceramics > Alumina (Typical) and drag and drop that material onto the Heated Block cuboid in the data tree.
