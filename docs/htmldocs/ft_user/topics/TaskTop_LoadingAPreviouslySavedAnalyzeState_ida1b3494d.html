Loading a Previously Saved Analyze State

Return to a previous configuration of results.

Prerequisites

  • A saved xml results state file.

Procedure

  1. Choose Analyze > Load State.

    The Load State dialog box is opened at solution_dir/project_dir/PDProject.

    The file may have been saved to a different directory.

  2. Select the file and click Open.

Results

Any scalar field and vector field settings are reinstated, and any plot, annotation, animation, saved table, and viewpoint definitions are reloaded.
