Exporting a Results Table

Creating a comma-separated text file containing results table data.

Procedure

  1. You have a choice:

    If you want to...

    Do the following:

    Only export the currently selected table.

    Right-click anywhere over the table cell area and choose Export Table.

    Export all tabbed tables in the current group.

    Right-click anywhere over the table cell area and choose Export All Tables.

    For transient analyses, export the table data for all saved time steps, or, in the case of Monitor Points, export the table data for all time steps.

    Right-click anywhere over the table cell area and choose Export Table (with Transient Data).

    The Export Table dialog box is opened.

  2. Navigate to a folder, optionally enter a filename, and click Save.

    The default filename is:

    project name - object name - table name.csv

    Where object name is only present if a single object was selected, and table name is a list of table names separated by commas if more than one table is exported.

    Reserved characters that might appear in the filename are replaced by underscores to comply with Windows filename conventions.

Results

If you chose to export all tables, the tables data are all contained within one worksheet. Each table is separated by a blank row.

If the length of the file pathname exceeds the Windows limit, then the file will not be exported and an E/6002 error message will be output.
