Overview of Grid Modification

User control over the dimensions of the spatial solution grid.

The following functionality enables you to modify the base grid to suit your needs:

  • Grid Constraint, either at system level or applied to individual objects, to constrain the sizes of grid cells or the number of cells.

  • Grid Localization, to restrict grid lines from extending outwards from objects to the edges of the solution domain.

  • Grid Inflation, to extend the grid beyond objects, either by a set distance or by a percentage of their size.

  • Grid Smoothing, to remove large variations in neighboring cell sizes and large aspect ratios between directions.

  • Deactivating keypoints, to remove grid lines associated with keypoints.
