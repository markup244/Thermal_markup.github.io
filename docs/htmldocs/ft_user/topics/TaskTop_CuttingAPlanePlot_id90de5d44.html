Cutting a Plane Plot

You can show the geometry clipped above or below a plane plot.

Procedure

  1. Select the plane plot.
  2. In the property sheet, at the Cut Plane property, select Above or Below.

Results

The model is cut at the plane plot such that it is not rendered above or below the plot.

You can cut more than one plane plot.
