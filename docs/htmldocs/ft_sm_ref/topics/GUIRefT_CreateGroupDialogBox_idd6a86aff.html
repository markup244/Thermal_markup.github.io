Create Group Dialog Box

To access: Select Create New from the dropdown options in the Groups tab of a property sheet.

Use this dialog to create and assign a new Aisle Group name to a Cooler, Rack, Recirculation Device, Fixed Flow or Perforated Plate, or a new Cooler Group name to a Cooler or Rack.

Objects

Field

Description

Group Name

A user-defined name.
