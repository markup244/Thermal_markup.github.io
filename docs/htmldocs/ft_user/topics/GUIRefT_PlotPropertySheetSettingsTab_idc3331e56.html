Plot Property Sheet Settings Tab

To access: Right-click in a plot, choose Edit Settings then select the Settings tab.

Use this property sheet tab to control the scaling and range of a profile plot axis, and to copy the plot data.

Objects

Field

Description

Plot Name

The name that appears in the title bar of the plot window.

Variable

(Monitor Points and Results v Distance plots) Sets the variable to be plotted from a dropdown list of available variables.

Grid Start X, Y, Z

(Results v Distance plots) The start point of the profile line. The x-axis (Distance) of the plot represents the length of the profile line.

The default start point is defined by the coordinates at the cell center in the low XYZ corner of the domain.

Grid End X, Y, Z

(Results v Distance plots) The end point of the profile line.

The default end point is defined by the coordinates at the cell center in the high XYZ corner of the domain.

Time Step

(Residuals v Iteration plots in Transient cases) Select the time step for which all the iterations are to be displayed from the dropdown list. The time step can be selected independently for each profile.

X Axis/Y Left Axis

Scale

Sets the axis scaling as either Linear or Logarithmic.

Lower Limit

Sets the start of the axis range:

  • Automatic — instructs the program to display data from the lowest value.

  • Manual — activates the value range input field to allow you to set the lowest value you want to see included in the plot.

  • % of Range — Activates the value range input field used to set the lowest value you want to see included in the plot as a percentage of the overall data range.

Upper Limit

Sets the end of the axis range:

  • Automatic — instructs the program to set the end at the highest value.

  • Manual — activates the value range input field to allow you to set the highest value you want to see included in the plot.

  • % of Range — activates the value range input field used to set the highest value you want to see included in the plot as a percentage of the overall data range.

Show Legend

Controls visibility of the legend in the plot.

Use Selection

(Monitor Points plots only) Click this button to show data only for the selected monitor points. The plots of unselected monitor points are removed from the chart.

To reinstate all the plots, choose Edit > Undo Modify Monitor Point Plot.Associated Geometry.

Selected monitor points

(Monitor Points plots only) A read-only field, which is updated to show the names of the monitor points selected by the Use Selection button.

Select Associated Geometry check box

(Monitor Points plots only)

Note:

The setting applies to all monitor point charts.

Check this check box if you want the monitor points associated with a monitor point chart to be selected whenever the chart is selected.

If the check box is left unchecked then selecting a monitor point chart does not automatically select the monitor points in the data tree or GDA.

Copy Data

Click this button to copy the plot data to the clipboard, see “Exporting a Plot Profile”.
