TEC Sizing

The relationship between TEC size (length and width, but not height) and its thermal performance is of interest to the designer.

There are two ways of that this relationship can be determined:

  • Swap a TEC with another held in the library. This is a more common approach.

  • Alter the size of the TEC using the Command Center, using linear functions to ensure the TEC is always centered on the component and to potentially alter the characterization data as a linear function of the size (if appropriate).
