Removing Selected Features

You can remove selectable features or faces to simplify the model.

Prerequisites

  • You are in Select mode and the Current Selection Mode is set to Feature or Face, see “Geometry Selection and View Manipulation”.

Procedure

  1. Select the feature or face.
  2. Click the Local Simplify icon or choose Tools > Local Simplify to open the Local Simplify dialog box.
  3. Click the Remove button or press the Delete key.
