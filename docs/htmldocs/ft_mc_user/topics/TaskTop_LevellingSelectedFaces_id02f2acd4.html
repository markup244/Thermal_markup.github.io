Leveling Selected Faces

To simplify the model, you can level selectable faces.

Prerequisites

  • You must be in Select mode and the Current Selection Mode must be Face, see “Geometry Selection and View Manipulation”.

Procedure

  1. Select one or more faces.
  2. Click the Local Simplify icon or choose Tools > Local Simplify to open the Local Simplify dialog box.
  3. Click the Level button or press L.
