Solving the Project

If you have followed the earlier sections, your model should now be ready for a solver run.

Prerequisites

  • Save the project.

Procedure

Either choose Solve > Solve or click the Solve icon.

A sanity check is performed first and the Message Window reports that an open external boundary does not have an ambient attached.



For now, ignore this since the default ambient will be sufficient for our purposes.

After the sanity check has been performed, the solution continues and the Profiles window opens.

Note:

Generally, it is good practice, before running a solve, to perform a sanity check by choosing Solve > Sanity Check. Message descriptions and troubleshooting information can be found in the Simcenter Flotherm User Guide.

Results

The solution completes to show converged plots.
