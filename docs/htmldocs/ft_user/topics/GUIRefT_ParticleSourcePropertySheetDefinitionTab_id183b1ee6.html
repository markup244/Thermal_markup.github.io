Particle Source Property Sheet - Definition Tab

To access: Expand the Particles node of the Results tree and select a particle source. The Definition tab opens by default.

Use this tab of the property sheet to define particle sources and particle streamlines.

Objects

Object

Description

Name

The name may be changed by overtyping.

On

Show/hide particle streamlines.

The particle source icon in the Results tree is dimmed when particle streamlines are not displayed.

Geometry

The name of the associated geometry.

Update

Click to move the particle sources to the currently-selected object.

Number of Streamlines

The number of sources or streamlines.

Visible Source

Show/hide particle sources.

Area Source

When unchecked, the sources are distributed around the edge of the source shape. When checked, the sources are distributed across the area of the source shape. If the source shape is a line, then this check box has no effect.

Source Shape

The shape of the area where sources are located. Choose from:

  • Ellipse — particles are released from an elliptical shaped source area.

  • Rectangle — particles are released from a rectangular shaped source area.

  • Line — particles are released along a line.

Manipulator Type

Choose one of the source manipulators to move the sources. You must be in Select mode for the manipulator to function.

  • None — No manipulator is active.

  • Translation — Activates the Translation manipulator.

  • Scale — Activates the Scale manipulator.

See “Creating and Moving Particle Sources”.

Source Plane Orientation

The plane in which particle sources are located: X, Y, or Z.

Particle Direction

Choose from:

  • Forward — Particles track away from their source.

  • Reverse — Particles track backward towards their source. This is useful for locating particle sources.

  • Both Directions — A combination of the above tracks: particles track away from and towards their source.

Scalar Field

The colors of particles are dependent on the values of the scalar variable selected here.

Vector Field

The sizes and directions of particles are dependent on the values of the vector variable selected here.

If Heat Flux or Current Density is selected, the particles flow through solid objects, even when those objects are hidden.

Min

Minimum value of the Scalar Field variable.

Max

Maximum value of the Scalar Field variable.

Create Minimum Annotation button

Click to add an annotation to the plot at the location of the minimum scalar field value.

Create Maximum Annotation button

Click to add an annotation to the plot at the location of the maximum scalar field value.
