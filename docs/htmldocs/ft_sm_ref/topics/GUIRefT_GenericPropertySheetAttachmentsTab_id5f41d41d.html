Generic Property Sheet Attachments Tab

To access: Select an object and then select the Attachments tab of the property sheet

Use this tab to attach or detach attributes to or from the selected object.

Objects

The attributes that can be attached depend on the selected object.

The attribute dropdown selection includes:

No Attachment

No attribute is attached.

Create New

You can create a new attribute.

Load From Library

You can load an attribute from a library.

A list of project attributes.

You can select an existing project attribute.

Usage Notes

Use the expansion buttons when you want to define different attributes in different directions or planes.
