Importing a Power List

Often, power information changes frequently throughout the board design process. Importing a power list in CSV format enables you to update a number of component power values at the same time.

Restrictions and Limitations

The import power list only enables you to modify power values, it does not allow you to add or remove components.

Prerequisites

A CSV file in Power List CSV file format.

Procedure

Choose File > Import > Import Power List and select the file.
