Overview of the Allegro Package Designer Interface

The interface enables users to directly send board outline, placement keep out areas, and component and package information from APD software.

The exported data can be:

  • used by Simcenter Flotherm PCB in a new or existing Simcenter Flotherm PCB project, or

  • imported into EDA Bridge for subsequent transfer into Simcenter Flotherm.

The interface also enables the user to import modified component placements made by Simcenter Flotherm PCB to improve the thermal performance of the board, back into the APD design.
