Modeling Non-Uniform Dissipation Using Discrete Sources

Non-uniform power dissipation is modeled by defining the power output at specified rectangular locations, termed discrete sources.

Procedure

  1. In the Die Construction property sheet, select Discrete Sources Source Type.
  2. Click the Source Definition Click to Edit button.

    The Non-Uniform Discrete Source Definition dialog box is opened.

  3. Select the desired units for Power Unit and Size Length Unit.
  4. Depending on the how you want to enter data, choose one of the following methods.

    If you want to...

    Do the following:

    Enter data manually.
    Note:

    By default, a single source, DiscreteSource1 of dimensions 1 mm × 1 mm and no specified power is created.

    1. Edit the default DiscreteSource1 row.

    2. If another source is to be defined, click the + button to add another row in the table.

    3. Enter another set of values.

    4. Continue adding data in this way.

      The chart is updated as you enter data. You can delete rows by clicking the - button.

    Enter data by CSV file import.

    1. Click Import CSV File.

    2. Navigate to, and select, a valid CSV file and click Open.
  5. Click OK to save the data and close the Non-Uniform Discrete Source Definition dialog box.

Results

  • The Total Power value in the Construction tab of the Die property sheet is updated to show the total calculated power.

  • The drawing board shows the layout of the sources on the Die surface, for example, see Figure 1.
