Importing and Exporting Libraries

An entire library can be exported to the file system for subsequent distribution to other EDA Bridge installations.

Library items are stored in PDML format, allowing full data integration to Simcenter Flotherm and FloTHERM PACK.

Procedure

  1. To export a library, right-click the library and choose Export to open a file browser.
  2. Name the file, which must have a .library extension, browse to a destination folder and click Save.
  3. The file can be e-mailed, for example, to another Simcenter Flotherm installation.
  4. To import a library, right-click an existing library and choose Import.
