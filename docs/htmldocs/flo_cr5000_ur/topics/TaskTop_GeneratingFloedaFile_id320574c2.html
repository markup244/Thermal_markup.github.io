Generating a FLOEDA File

Run the interface command with previously-generated ASCII input files.

Procedure

Assuming the file names generated by “Generating ASCII Input Files”, enter the following ZukenBrdDesToEDAI command:
ZukenBrdDesToEDAI –job xyz -cov image –out xyz.floeda

Results

The following output will be generated as the interface runs:
