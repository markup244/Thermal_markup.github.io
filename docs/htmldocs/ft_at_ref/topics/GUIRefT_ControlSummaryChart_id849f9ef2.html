Control Summary Chart

To access: Open a Control attribute property sheet then click the Control Summary Click To View button.

Use this chart to examine the set of defined control curves, and to export the data points of those curves.

Description

Low and high temperature thresholds are shown on control curves by blue and red squares respectively
