Source Attribute Power vs Temperature Chart

To access: Open a Source attribute property sheet, click the Activate check box, select an Option of Non Linear Source, then click the Power vs Temperature Chart Click To Edit button.

Use this to chart to define or examine the non-linear relationship between power and temperature of the source.

Objects

Field

Description

Temperature

The units of time values entered into the table.

Power

The units of power values entered into the table.
