Creating PCBs

PCB SmartParts are created as children of assemblies. By default, a non-conducting PCB is created, one-tenth the size of the solution domain in the X- and Y-directions.

Note:

Use EDA Bridge to construct detailed models of mother boards with the ability to import library components.

Procedure

  1. Select an assembly in the data tree.
  2. In the New Object Palette, choose Project Manager Create or Drawing Board Create and double-click the PCB icon.

Results

Components can be added as SmartParts, either as PCB Components or Compact Components.
