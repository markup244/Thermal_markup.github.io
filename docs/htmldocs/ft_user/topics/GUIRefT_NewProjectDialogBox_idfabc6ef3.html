New Project Dialog Box

To access:

  • Ctrl+N.

  • Project > New.

Use this dialog box to create a new project from an existing template. You can also set the default template which is loaded when you start Simcenter Flotherm.

Description

If there are outstanding edits to the current project, you are given the option to either Save or Discard these before the New Project dialog box is opened.

The dialog box consists of a series of tabbed panels which each display a list of project templates to choose from. The current, default project type is selected when the dialog box opens.

The templates are organized in categories, each category has a separate tab.

The icons indicate whether templates have been saved with or without results by the presence or absence of a thermal plot.

Objects

Object

Description

Application Examples tab (where installed)

Practical examples of Simcenter Flotherm modeling techniques.

By default, this tab and the application examples that it contains are not created during installation but you can create it and download the application examples by following the steps in “Installing the Application Examples”.

Defaults tab

DefaultSI defines a space of 27 m3. DefaultUS defines a space of 1000 ft3. In both defaults, the space is filled with air at 20°C 50% RH and pressure of 1 atmosphere.

atx tab

Standard form factor template for ATX PC designs.

btx tab

Standard form factor template for BTX PC designs.

Set Default

Click to make the selected template the default template when Simcenter Flotherm opens.

Delete

Click to delete the selected template.
