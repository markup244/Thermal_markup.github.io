Localization Near the Edge of the Solution Domain

If a localized grid space is one grid cell away from the edge of the solution domain, then an additional grid line is inserted to ensure there are two grid cells.

This effect of localization is shown in Figure 1. The addition of the extra grid cells ensures an accurate CFD solution between the localized grid space and the edge of the domain.

Figure 1. Localization Near the Solution Domain Edge

Note:

An extra grid line will always be inserted under such circumstances, regardless of any minimum cell size tolerance defined. Care should, therefore, be taken to ensure a localized grid space is exactly coincident with the edge of the domain, if that is what is required.
