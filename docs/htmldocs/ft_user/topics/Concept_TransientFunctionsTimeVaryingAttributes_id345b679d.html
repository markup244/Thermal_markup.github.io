Transient Functions (Time-Varying Attributes)

You can set time-varying Thermal, Source, or Ambient attributes by attaching a Transient attribute that modifies the host attribute by a multiplication factor.

These Transient Functions can be overlaid on the Time Step Distribution Plot. When selected, these functions are displayed in red, and the multiplication factor is identifiable by reading the right-hand axis.

Figure 1. Transient Function Overlay

Such overlays provide help for setting up time steps. In this example, a combination of Uniform, Symmetrical Power and Increasing Power distribution types have been used to more closely monitor changes due to the transient function that is being applied.
