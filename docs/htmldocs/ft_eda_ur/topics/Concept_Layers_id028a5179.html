Layers

A group of four layers are provided by default with the Motherboard or a new Daughterboard. More layers can be added.

On the Motherboard, the default layers are named Signal, Power or Ground, Power or Ground, and Signal. On the DaughterBoard, the default layers inherit the name of the DaughterBoard.

The first and last layers are aligned with the top and bottom of the board respectively, and the two intermediate layers are spaced evenly across the thickness of the board, as shown in Figure 1.

Figure 1. Default MotherBoard Layers

A new layer is added below the selected layer. By default, a new layer is a Metallic layer, but this can be changed, using the Layer property sheet, to a Dielectric layer. Dielectric layers are represented by green icons as shown in Figure 2.

Figure 2. Icon Representation of Metallic and Dielectric Layers in the Data Tree

To control the spacing of the layers, uncheck the Equispaced check box off in the Layer Group property sheet and enter offset values for each layer.

Layer Patches can be added to metallic layers.

Electrical Vias can be added to dielectric layers.

Modeling Layers

  • Typical signal layer parameters are 20% coverage, 0.034 mm (1 oz) thickness.

  • Typical power or ground layer parameters are 90% coverage, 0.017 mm (0.5 oz) thickness.
