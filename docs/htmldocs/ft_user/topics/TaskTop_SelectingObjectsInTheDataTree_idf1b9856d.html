Selecting Objects in the Data Tree

Individual multiple selection and contiguous multiple selection are supported.

Procedure

  1. You have a choice.

    If you want to...

    Do the following:

    Select multiple objects individually, that is, one at a time.

    • Select the first item and then press Ctrl+left-click over each additional required data item.

      If your design has a number of primitives/SmartParts/regions/cutouts/monitor points under the root assembly, you can select a group of them of the same or of mixed type.

    Select multiple objects in a contiguous group.

    • Select the first object, hold the Shift key, and then select the last object.

      Alternatively, you can select the last object, hold the Shift key, and then select the first object.
  2. To deselect an item or items, click in the blank area of the display or press Esc.
