Downloading a Found Library Item

A quick way of downloading a library item to a selected project part, without the need to expand the library.

Prerequisites

  • In the project tree, select a valid part to receive the library item

Procedure

  1. Find items by Searching for a Library Item
  2. Double-click the item in the Found List.
