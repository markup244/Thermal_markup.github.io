Creating Time Patches

Time steps are created by adding and defining time patches. A time patch is made up of one or more time steps.

Procedure

  1. Choose Model Setup > Transient Solution to open the Transient Solution dialog box.
  2. Use the Time Patches pane to create and configure time patches.

    If you want to...

    Do the following:

    Use transient functions.

    Click AutoCreate to automatically create time patches based on those functions currently attached to the project.

    If there are none, then a single time patch is created. If there are transient functions, then time patches are created for each period (if the function is periodic) or for each subfunction (if the function is subdivided).

    The time patches are also divided into time steps dependent on the settings in the Number of Steps Control and Time Step Distribution frames.

    Adjust start/end times or edit time patch names.

    Add time patches by clicking the plus (+) button, adjust start/end times if necessary, and edit time patch names by selecting them.

    Modify a time patch.

    Select the patch.

    Note:

    Selected time patches are colored green in the Time Step Distribution Plot.

    The parameters on the right of the patch list are those applicable to the currently selected patch.

    Select multiple time patches.

    Select the time patches, and press Ctrl+select or Shift+select.

    Note:

    When you select more than one time patch, data common to all of the selected time patches is shown, and changes made to common fields affect all selected time patches.

    Data fields that are not common are unavailable, and the values particular to the time patches are maintained.
  3. When you have finished, click OK to close the Transient Solution dialog box.
