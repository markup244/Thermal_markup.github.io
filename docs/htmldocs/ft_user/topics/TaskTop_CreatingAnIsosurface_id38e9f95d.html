Creating an Isosurface

An isosurface plot is a surface joining all points which have the same value for a given variable.

Prerequisites

  • The Project Manager is in Analyze mode.

Procedure

  1. Click the Create Isosurface icon
  2. You can customize the plot by changing values and settings in the plot property sheet.

Results

The first isosurface plot created is named Isosurface. Subsequent plots are named Isosurface n.
