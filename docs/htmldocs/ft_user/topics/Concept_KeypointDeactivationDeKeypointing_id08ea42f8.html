Keypoint Deactivation (De-Keypointing)

Keypoint deactivation occurs when objects are close together to reduce an excessive number of grid cells.

Keypoint deactivation occurs when a dimension of an object is smaller than the Minimum Grid Size set in the System Grid property sheet. In such cases no grid line coincides with the keypoint.

Figure 1 shows an example where deactivation is invoked on one cuboid, but not on another. The Z dimension of the lower cuboid is greater than the minimum grid size.
