Using Parallel Processing

To speed up the solving process, provided you have multiple processors, you can specify more than one processor when running the solver.

Restrictions and Limitations

Parallel processing is only supported on shared memory systems.

Procedure

  1. Open the User Preferences dialog box by clicking the User Preferences icon or choosing Edit > User Preferences.
  2. In the Number of Processors to Use field, enter the number of processors you want to use.

Results

You can check that multiple processors are being used by monitoring your computer using the Windows Task Manager when running the solver.
