Detaching an Attribute

You must detach an attribute before it can be deleted.

Procedure

  1. Find all objects that have the attribute attached.
  2. For each object, open the Attachments tab of the object property sheet.
  3. Select either No Attachment or a different attribute from the attachments dropdown list.
