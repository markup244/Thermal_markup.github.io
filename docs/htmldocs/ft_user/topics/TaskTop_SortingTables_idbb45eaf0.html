Sorting Tables

Table sorting is not enabled by default.

Procedure

  1. Select the header of the column by which the table is to be sorted.
  2. Right-click and choose Enable Sorting.

    The sort descending icon is displayed to the right of the header title indicating that the table has been sorted with this column in descending order (high to low).

  3. To sort the table by a different column, click the header of that column.

    The sort descending icon moves to the newly-selected column and the table is now sorted with this column in descending order.

  4. To sort in ascending order (low to high), click the header of the column that is the current sort-by column.

    The sort ascending icon is displayed.

  5. To restore the default order, right-click in any column header and choose Disable Sorting.
