User Preferences Dialog Box - Analyze Mode Tab

To access: F11, or choose Edit > User Preferences, or click the User Preferences tool icon, then select the Analyze Mode tab.

Use this tab to specify the resolution of GDA snapshots, change font sizes when the GDA is in Analyze mode, and change the rendering of animations.

Objects

Object

Description

Output Snapshot use Screen Resolution

Use the screen resolution when creating a GDA snapshot.

Snapshot Width

(Output Snapshot use Screen Resolution unchecked) User-specified resolution width, in pixels.

Snapshot Height

(Output Snapshot use Screen Resolution unchecked) User-specified resolution height, in pixels.

Ambient Light Level

Sets the intensity of the minimum light falling on the model. Most noticeable when viewing 3D Isosurface plots.

Legend Font Size

Controls the size of the legend font in the GDA views when in Analyze mode.

Title Font Size

Controls the size of the project name in the GDA views when in Analyze mode.

Show Title

Shows/Hides the project name in the GDA views when in Analyze mode.

Animation

Number of Frames

Total number of frames in an animation. Default 18.

Frames per Second

Speed of an animation in frames per sec (fps). Default 30.

Note:

It may not be possible to render the plot at the selected fps, it is a target value.

Swing

Sets animations to loop continuously in alternate directions. The plot location or variable values travel from one side of the animation range to the other. When it reaches the end of the range, it reverses its direction and travels back towards the start of the range.

Display Subdomains

Display subdomains, in orange.

Default unchecked.

Note:

This can also be set on the Drawing Board tab (see “User Preferences Dialog Box - Drawing Board Tab”).

Display Solution Domain

Display the solution domain, in yellow.

Default unchecked.

Note:

This can also be set on the Drawing Board tab (see “User Preferences Dialog Box - Drawing Board Tab”).

Display Geometry Without Expanding Tree

When unchecked, geometry of assemblies that are collapsed in the data tree is not shown in the GDA - instead, an outline of the assembly is shown (as a green wireframe box). When checked, geometry of all assemblies is shown in the GDA, regardless of whether they are expanded or collapsed in the data tree. Default unchecked.

Gradient Background

When in Analyze mode, sets the background of the GDA to be merged from the top background color to the bottom background color. If unchecked, the background is a uniform color.

For the GDA in Create mode, use the equivalent option in the Drawing Board tab of this dialog box.

Background Color

When in Analyze mode, the background color of the GDA.

For the GDA in Create mode, use the equivalent option in the Drawing Board tab of this dialog box.

Top Background Color

(Gradient Background) When in Analyze mode, the background color at the top of the GDA.

For the GDA in Create mode, use the equivalent option in the Drawing Board tab of this dialog box.

Bottom Background Color

(Gradient Background) When in Analyze mode, the background color at the bottom of the GDA.

For the GDA in Create mode, use the equivalent option in the Drawing Board tab of this dialog box.

Number of Significant Figures

The number of significant figures, between 1 and 7, displayed in results tables, annotations, and plot legends. Default 3.

Marginal Temperature Range

Applicable to EDA Components when reporting results. The margin, in degC, between the calculated junction or case temperature and a specified maximum which reports a Grade as “Marginal”, see “EDA Components Results Table”. Default 5 degC.

Usage Notes

You must close the dialog box to effect any changes made.
