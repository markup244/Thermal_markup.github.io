Using the Allegro Interface

This section describes the operational menu options and customizing dialog boxes.
Note:

The file browser is shared by the Load into existing Simcenter Flotherm PCB project, Export .floeda file and Import Placement Changes menu options and therefore the last directory used by any of those options will be the directory which will be the starting point for the next option used. The starting directory is reset every time a new design is loaded.
