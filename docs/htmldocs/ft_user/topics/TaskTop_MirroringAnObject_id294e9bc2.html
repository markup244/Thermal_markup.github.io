Mirroring an Object

A mirrored copy can be made of an object or group of objects, using the workplane as the mirror plane.

Procedure

  1. Use the Workplane toolbar controls to create a mirror plane midway between the object(s) to be copied and the location of the required object.
    Figure 1. Objects Before Mirroring

  2. Select the object(s) and choose Geometry > Mirror, or click the Mirror icon in the Drawing Board toolbar.

Results

A mirrored copy of the selected object(s) is made.

Figure 2. Objects After Mirroring
