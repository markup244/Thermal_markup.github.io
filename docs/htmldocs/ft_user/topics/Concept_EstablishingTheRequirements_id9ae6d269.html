Establishing the Requirements

Take the time to assess exactly what to model before starting.

A sketch that shows the major features and dimensions will help to start the case. This will allow you to decide how much needs to be included and also to orient yourself with the Simcenter Flotherm axis set.

At the outset identify the important key features of the model. It is in these areas that you may have to concentrate more grid cells to increase the resolution of the results. Also place monitor points into key areas. These allow the variables to be tracked at particular locations during the solution process. This will quickly give feedback as to whether the results are sensible and increase confidence when leaving a simulation running for an extended period.
