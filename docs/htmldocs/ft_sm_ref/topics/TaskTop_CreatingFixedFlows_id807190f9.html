Creating Fixed Flows

Fixed Flow SmartParts are 2D rectangles that represent apertures through which fluid flows.

Procedure

  1. Select an assembly.
  2. In the New Object Palette, choose Project Manager Create or Drawing Board Create and double-click the Fixed Flow icon.

Results

A fixed flow is represented in the drawing board as a 2D rectangle with a flow direction arrow. If the flow direction arrow is not visible, check that it has not been suppressed (User Preferences).
