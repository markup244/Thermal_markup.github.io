Model Setup Tab

To access: Click the Model Setup tab.

Use this tab to define the principal features of the mathematical model.

Objects

Field

Description

Type of Solution

Set the type of solution to determine the variables solved. There are three options: Flow and Heat Transfer, Flow Only, and Conduction Only. Refer to “Types of Solution”.

Dimensionality

Sets the overall solution domain to either two or three dimensions. See “Modeling Dimensionality”, which discusses the importance of this setting and how it affects the solution.

Radiation

Choose the radiation modeling type for boundaries to which you have attached radiation attributes from the Radiation popup menu. There are three options:

  • Radiation Off — The appropriate modeling choice for most forced convection systems and initial simulation of all types.

  • Radiation On — Calculates the view factors between all pairs of surfaces for which radiation has been activated. If the path between the surfaces is blocked, then the view factor is calculated based upon the proportion of radiation that can pass.

  • Radiation On - High Accuracy — Calculations are more accurate. The calculation of the radiation exchange factor could take up to four times longer.

See Radiation Attribute Property Sheet in the Simcenter Flotherm Project Attributes Reference Guide.

Joule Heating

Include Joule Heating calculation in the solution. When Joule Heating is enabled:

  • Potential can be selected as a Source Type in a Source attribute.

  • Potential can be selected as a Variable in the Solver Control tab.

See “Joule Heating Modeled by Source SmartParts”.

Solar Radiation

Switches on solar radiation calculation to include in the solution. Click Click to Edit to open the Solar Configuration Dialog Box.

Transient Solution

Sets the solution to be a Transient solution, rather than the default Steady-State solution. See “Transient Analysis”. Click Click to Edit to open the Transient Solution Dialog Box.

Fluid

The Fluid attribute defining the circulating fluid (normally air) in the solution domain.

Gravity section — refer to “Model Setup Tab - Gravity Section”.

Turbulence section — refer to “Model Setup Tab - Turbulence Section”.

Global System Settings section — refer to “Model Setup Tab - Global System Settings Section”.

Capture Index

Calculate

Activate this option to request the calculation of the Capture Index (CI).

Sub-Groups

(Calculate) Enables attachment of Aisle Sub-Group names.

Stored Variables section — refer to “Model Setup Tab - Stored Variables Section”.

Auxiliary Variables section - refer to “Model Setup Tab - Auxiliary Variables Section”.

Usage Notes

Radiation

Note:

For geometrically complex situations, activation of full radiation will require more computational effort to calculate the exchange factors.

For more information on the strategy of radiation modeling, see Radiation Model in the Simcenter Flotherm Background Theory Reference Guide.
