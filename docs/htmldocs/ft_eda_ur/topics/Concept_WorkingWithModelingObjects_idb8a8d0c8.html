Working With Modeling Objects

Selecting an item in the node tree or the graphical 2D view results in valid geometry icons in the New Geometry toolbar becoming active.

When you click an active icon, the new object is added to the data tree, under the currently selected object.

By default, new objects become selected on creation, with their property sheet open. If you do not want objects selected on creation, then deactivate Select Objects when Created in the GUI Preferences dialog box.
