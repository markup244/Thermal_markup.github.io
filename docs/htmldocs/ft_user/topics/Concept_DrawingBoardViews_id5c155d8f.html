Drawing Board Views

The drawing board has four view configurations. The view configuration is independent between Create and Analyze modes.

  • Examiner View — for examining details, the single pane fills the drawing board.

  • Grid View — four panes. By default, these show X, Y and Z plane third-angle projections and an isometric view.

  • Split Horizontally — two panes arranged horizontally.

  • Split Vertically — two panes arranged vertically.

All views show the grid if the grid visibility switch (G) is on.
