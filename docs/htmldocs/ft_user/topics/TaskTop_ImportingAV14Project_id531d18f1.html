Importing a V1.4 Project

Version 1.4 files require a non-standard import treatment.

Prerequisites

  • A Simcenter Flotherm V1.4 *.prb or *.PRB file.

Procedure

  1. In the top of the data tree, right-click the project node and choose Import Project > V1.4 File.

    A file selection dialog box is displayed to find the required file.

  2. Select the V1.4 Project file that you want to import, then click Open.
