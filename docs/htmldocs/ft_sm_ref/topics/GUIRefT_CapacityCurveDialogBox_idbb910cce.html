Capacity Curve Dialog Box

To access: In the Construction tab of a Cooler property sheet, click the Capacity Curve Click to Edit button. This button is only visible when Capacity Limit is set to Variable.

Use this dialog box to define a cooler capacity curve from a sequence of points.

Objects

Field

Description

Temperature Unit

The units of Return Air Temperature used.

Power Unit

The units of Sensible Cooling Capacity used.

Table

Pairs of values of Return Air Temperature and Sensible Cooling Capacity.
