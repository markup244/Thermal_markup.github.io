Hiding Results Table Columns

Tidy up large results tables by hiding unwanted columns.

Procedure

You have a choice:

If you want to...

Do the following:

Hide selected columns.

  1. Select the column(s) that you want to hide.

  2. Right-click in the table column header and choose Hide Selected Columns.

Hide unselected columns.

  1. Select the column(s) that you do not want to hide.

  2. Right-click in the table column header and choose Hide Unselected Columns.

Results

The columns are removed from the table.

To reveal hidden columns, right-click anywhere within the table and choose Reveal Hidden Columns.
