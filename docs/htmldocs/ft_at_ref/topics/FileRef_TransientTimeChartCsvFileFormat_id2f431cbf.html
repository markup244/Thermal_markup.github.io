Transient Time Chart CSV File Format

A comma-separated text data file.

Use this file to set up a transient attribute profile which varies with time.

Format

A Transient Time Chart CSV File must conform to the following formatting and syntax rules:

  • The units are those currently set by the chart.

  • The list can be any length.

  • The values are in floating point format.

  • The file format is two columns of data separated by a comma.

    <time>,<multiplier> 
<time>,<multiplier> 
...

Parameters

time

A time at which you want to define a multiplier.

multiplier

The multiplier value at that time.
