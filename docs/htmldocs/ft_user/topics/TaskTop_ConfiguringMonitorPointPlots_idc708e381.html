Configuring Monitor Point Plots

By default, all monitor points are plotted in a newly-created monitor point plot chart. You can reduce the number of plots shown.

Prerequisites

  • You have created a monitor point profiles chart, see “Creating New Plots”.

Video

Learn how to modify a monitor points chart to only include plots for specific monitor points, and how to use the Select Associated Geometry check box to synchronize monitor point selection with a chart.

Procedure

  1. Select the monitor points that you want to be displayed in the profiles plot.

    Selection can be done in the plot itself or in the data tree or GDA.

  2. Right-click in the plot and choose Edit Settings.

    The Settings tab is opened.

  3. Click Use Selection.

    This button is in the Monitor Points section of the tab.

  4. Optionally, check the Select Associated Geometry check box (this is on the far-right of the Monitor Points section) if you want the monitor points associated with the chart to be selected whenever the chart is selected.

Results

Plots of unselected monitor points are removed from the chart.

The names of the selected monitor points appear in the read-only field to the right of the Use Selection button.
