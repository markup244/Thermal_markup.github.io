Geometry Files

Simcenter Flotherm can import and export geometry in different file formats.
  • PDML Files
  • FloXML Files
  • FloFEA Files
  • ECAD File Formats
  • ECXML Files
  • JEDEC Part Model XML Files (ThermalSection)
  • Files Imported via MCAD Bridge
  • Files Imported via EDA Bridge
  • Power Map Files
