Setting Save Times

When setting up a transient solution, you can optionally decide to save 3D data at specific times (Save Times) during the transient solver run.

Procedure

  1. Choose Model Setup > Transient Solution to open the Transient Solution dialog box.
  2. In the Save Times pane.

    If you want to...

    Do the following:

    Create a set of Save Times based on the value entered in Every Nth Step.

    Click Populate.

    Note:

    If you have entered a number greater than the total number of steps, only the time of the first step is saved.

    Add the time entered in the Value field to the list.

    Click Add.

    The value snaps to that of the nearest time step. For example, if the overall transient is 100 s long with time steps set at intervals of 20 s, attempting to add a Save Time at 38 s will create one at 40 s.

    Note:

    You cannot save data at any other times than the ends of time steps.

Results

The list of Save Times is updated as you use the Populate, Add, Delete, and Delete All buttons.
