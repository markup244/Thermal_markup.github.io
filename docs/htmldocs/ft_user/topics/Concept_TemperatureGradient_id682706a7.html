Temperature Gradient

Temperature gradient values are not normally stored, but can be if the GradT check box in the Stored Variables section of Model Setup tab is checked.

When these values are stored, they are viewable in Analyze mode.

  • Grad T is a vector indicating the size and direction of the steepest temperature gradient calculated from cell centers. It is stored as three scalar field components: X Grad T, Y Grad T, and Z Grad T.

  • Mag Grad T, the magnitude of the temperature gradient, is a scalar field stored at cell centers.
