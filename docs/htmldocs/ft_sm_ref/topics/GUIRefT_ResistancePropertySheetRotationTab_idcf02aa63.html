Resistance Property Sheet Rotation Tab

To access: Only available for collapsed resistances and when Local coordinates are selected. Select a Resistance SmartPart in the model and, in the Location tab, select a collapse face.

Use this property sheet to define an angle of rotation of a collapsed resistance.

Objects

Field

Description

Rotate

The angle of rotation must be between -90 and 90 degrees. A positive angle of rotation is a clockwise rotation when viewed along the axis from the origin in the positive direction.

About

The axis of rotation:

  • None — no axis, that is, ignore the angle of rotation.

  • The two axes that are not the Collapse axis. For example, if the resistance has been collapsed to a Yo face, then rotation is possible about the Xo or Zo axis.
