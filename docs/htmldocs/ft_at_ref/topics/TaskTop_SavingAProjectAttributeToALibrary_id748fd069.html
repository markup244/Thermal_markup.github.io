Saving a Project Attribute to a Library

A project attribute saved to a library ensures that the attribute is available for other projects.

Procedure

  1. Open the Library tree.
  2. Select a Library folder.
  3. Open the Project Attributes tree.
  4. Right-click an attribute and select Save To Selected Library.

Results

Expand the Library to view the copied attribute.
