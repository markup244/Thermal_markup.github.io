TEC Property Sheet

To access: Select a TEC SmartPart in the model.

Use this property sheet to configure a TEC.

Description

The Hot Side 1 and Hot Side 2 tabs are used to enter data at two hot side temperatures. This data is usually available from manufacturer’s data sheets.
